## Code Compliance

### IBC 2021 Mass Timber Types
| Type | Stories | Area (sf) | Fire Rating |
|------|---------|-----------|-------------|
| Type IV-A | Up to 18 | Unlimited | 3-hour |
| Type IV-B | Up to 12 | Varies | 2-hour |
| Type IV-C | Up to 9 | Varies | 2-hour |
| Type IV-HT | Per table | Per table | 1-hour (HT) |

### Fire Resistance
| Rating | Char Depth | Notes |
|--------|------------|-------|
| 1-hour | ~1.5" | Char rate 1.5"/hr |
| 2-hour | ~3" | Additional layers |
| 3-hour | ~4.5" | Requires protection |

### Char Rate Calculation
```
Char depth = 1.5 × (time in hours) × 1.2 (factor of safety)

Example: 2-hour rating
Char = 1.5 × 2 × 1.2 = 3.6" required sacrificial wood
```

## Construction Process

### Prefabrication
| Element | Shop Work |
|---------|-----------|
| CLT panels | Cut to size, openings |
| Glulam | Cut, drilled, hardware |
| Connections | Pre-installed where possible |
| MEP | Coordinated penetrations |

### Installation Sequence
```
1. Foundation/podium complete
2. Columns erected (crane)
3. Beams placed and connected
4. CLT floor panels placed
5. CLT walls erected
6. Repeat for each floor
7. Roof structure
8. Enclosure (critical path)
```

### Crane Requirements
| Element | Weight | Crane Capacity |
|---------|--------|----------------|
| CLT panel | 50-150 psf | 10-30 tons |
| Glulam beam | 35 pcf | Varies |
| Column | 35 pcf | Varies |
| Pick capacity | Per lift plan | 1.25× factor |

### Schedule Advantages
| Factor | Savings |
|--------|---------|
| Prefabrication | 20-40% faster |
| Dry construction | Weather tolerance |
| Fewer trades | Less coordination |
| Reduced finishing | Exposed structure |
