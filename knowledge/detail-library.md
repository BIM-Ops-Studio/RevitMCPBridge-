# Detail Library - Common Conditions

## Window Details

### Window Head (Typical)
```
Conditions to show:
- Lintel/header above (steel, wood, or masonry)
- Flashing at head (through-wall or surface)
- Sealant joint
- Window frame
- Interior trim/return
- Insulation continuity
```
**When needed:** Every window type in schedule

### Window Jamb (Typical)
```
Conditions to show:
- Rough opening to frame
- Shims and sealant
- Backer rod and sealant (exterior)
- Interior trim
- Weather barrier lapped to frame
```

### Window Sill (Typical)
```
Conditions to show:
- Sill slope (1/4" min pitch to exterior)
- Sill flashing (pan flashing)
- Sub-sill or wood sill
- Interior stool and apron
- Sealant locations
```
**Critical:** Sill flashing is #1 leak point

### Window at Stucco
```
Special items:
- J-mold or casing bead at perimeter
- Control joint if span > 10'
- Weep screed at sill
- Flexible sealant at frame
```

## Door Details

### Exterior Door Threshold
```
Conditions to show:
- Threshold type (mill, oak, ADA)
- Weather stripping
- Sill pan (required under threshold)
- Sealant at jamb/sill intersection
- Floor finish transition
```
**ADA Note:** 1/2" max threshold height

### Exterior Door Head
```
Conditions to show:
- Header/lintel
- Flashing
- Drip edge or head flashing
- Sealant joint
- Weather stripping
```

### Interior Door Frame
```
Conditions to show:
- Frame to rough opening
- Shims
- Casing or trim
- Drywall return (if frameless)
```

### Fire-Rated Door Frame
```
Special items:
- Frame type (hollow metal, wood)
- Label location
- Head and jamb details
- Closer location
- Smoke seal if required
```

## Roof Details

### Roof-to-Wall (Low Roof to High Wall)
```
Conditions to show:
- Counter-flashing (reglet or surface)
- Base flashing (up wall min 8")
- Cant strip at base
- Roof membrane termination
- Sealant at reglet
```
**Critical:** Most common leak location

### Parapet Detail
```
Conditions to show:
- Coping (metal, precast, stone)
- Coping slope (toward roof)
- Through-wall flashing
- Membrane termination
- Blocking for attachment
- Interior/exterior wall finish
```

### Roof Edge/Fascia
```
Conditions to show:
- Drip edge or gravel stop
- Fascia board/metal
- Soffit
- Ventilation (if applicable)
- Gutter attachment
```

### Roof Penetration
```
Conditions to show:
- Pipe boot or curb
- Flashing (base and counter)
- Sealant locations
- Height above roof surface
```

### Valley Detail
```
Conditions to show:
- Valley flashing (metal)
- Underlayment overlap
- Shingle or tile termination
- Ice and water shield (if applicable)
```

## Wall Details

### Base of Wall (Slab-on-Grade)
```
Conditions to show:
- Sill plate (PT wood)
- Anchor bolts
- Sill seal (foam gasket)
- Termite shield (if metal)
- Slab edge
- Flashing or weep
- Grade line
```

### Base of Wall (CMU)
```
Conditions to show:
- CMU to footing connection
- Vertical rebar from footing
- First course grouted solid
- Flashing at grade
- Waterproofing (if below grade)
```

### Wall Section at Floor
```
Conditions to show:
- Floor framing connection
- Fire blocking
- Rim joist
- Exterior sheathing
- Weather barrier continuity
- Insulation continuity
```

### Control Joint in Masonry
```
Location rules:
- Max 25' O.C.
- At openings (within 4' of corners)
- At changes in wall height
- At changes in wall thickness

Detail shows:
- Sealant and backer rod
- Building paper or bond breaker
- Rebar discontinuous through joint
```

## Interior Details

### Base Cabinet at Wall
```
Conditions to show:
- Cabinet to wall
- Toe kick
- Counter to wall
- Backsplash
- Upper cabinet
- Blocking for upper cabinets
```

### Ceiling at Wall (Soffit)
```
Conditions to show:
- Ceiling framing
- GWB termination
- Trim or transition
- Blocking if needed
```

### Stair Section
```
Conditions to show:
- Tread depth
- Riser height
- Nosing profile
- Stringer
- Handrail (height, extension)
- Guard (height, baluster spacing)
```

### Shower/Tub at Wall
```
Conditions to show:
- Cement board (not GWB)
- Waterproof membrane
- Tile and thin-set
- Shower pan (if applicable)
- Transition to floor
- Flange or surround edge
```

## Flashing Details

### Through-Wall Flashing
```
Use at:
- Base of masonry walls
- Shelf angles
- Window heads in masonry
- Anywhere water could penetrate

Shows:
- Metal flashing (min 24 ga galv)
- Drip edge extending past face
- End dams at terminations
- Weeps above flashing
```

### Sill Flashing (Pan Flashing)
```
Required at:
- All window sills
- All door thresholds
- Deck ledger connections

Shows:
- Slope to exterior
- Up-turned ends
- Back dam
- Drainage path
```

### Kick-Out Flashing
```
Required at:
- Roof-to-wall where roof ends at wall
- Prevents water running behind siding

Shows:
- Flashing extends past roof edge
- Diverts water into gutter
- Ties into step flashing
```

## When to Include Each Detail

### Every Project
- Typical wall section
- Foundation detail
- Roof edge detail
- Window head/jamb/sill

### As Applicable
- Stair section (if stairs)
- Parapet (if parapet roof)
- Expansion joint (large buildings)
- Fire-rated assembly (if rated walls)

### Complex Projects
- Custom details for unique conditions
- Enlarged plans at complex areas
- 3D isometric for complex flashing

## Detail Drawing Standards

### Scale
| Detail Type | Typical Scale |
|-------------|---------------|
| Wall section | 3/4" = 1'-0" |
| Window/door | 1-1/2" = 1'-0" |
| Flashing | 3" = 1'-0" |
| Small detail | 3" = 1'-0" or larger |

### What to Include
- All materials labeled
- Dimensions where critical
- Notes for installation
- Reference to spec section
- Indication of fire-rating if applicable
