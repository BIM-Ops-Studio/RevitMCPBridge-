# Senior Living Design

## Continuum of Care Levels

### Care Types
| Level | Description | Staffing |
|-------|-------------|----------|
| Independent Living (IL) | Active seniors, minimal services | Minimal |
| Assisted Living (AL) | Daily living assistance | 24-hour |
| Memory Care (MC) | Dementia, Alzheimer's | Specialized 24-hour |
| Skilled Nursing (SNF) | Medical care, rehabilitation | Medical staff |
| Continuing Care (CCRC) | All levels on one campus | Varies |

### Typical Resident Profile
| Level | Age | ADL Needs |
|-------|-----|-----------|
| Independent | 70-85 | None to minimal |
| Assisted | 80-90 | 2-3 ADLs |
| Memory Care | 75-95 | Cognitive support |
| Skilled Nursing | 80+ | Extensive medical |

## Unit Design Standards

### Independent Living Units
| Element | Requirement |
|---------|-------------|
| Entry door | 36" minimum |
| Interior doors | 32" clear minimum |
| Hallways | 42" minimum |
| Kitchen | Full, accessible |
| Bathroom | Walk-in shower preferred |
| Washer/dryer | In-unit or nearby |
| Emergency call | Required |
| Balcony/patio | Preferred |

### Assisted Living Units
| Element | Requirement |
|---------|-------------|
| Entry door | 36" minimum |
| Bathroom | Roll-in shower, grab bars |
| Kitchenette | Optional (sink, microwave) |
| Emergency call | Pull cord + pendant |
| Lighting | Adequate task lighting |
| Flooring | Non-slip, low pile |
| Furniture | Resident or provided |

### Memory Care Units
| Element | Requirement |
|---------|-------------|
| Simplified layout | Reduce confusion |
| Bathroom | Visible from bed |
| No kitchenette | Safety |
| Emergency call | Staff alert only |
| Lighting | Circadian rhythm support |
| Windows | Secured, limited opening |
| Doors | Concealed from wandering |
