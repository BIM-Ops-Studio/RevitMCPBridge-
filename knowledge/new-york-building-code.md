# New York Building Code - Key Amendments

**Based on**: NYC Building Code (2022), NYS Building Code (2020)
**Base Code**: IBC 2015 (NYC), IBC 2018 (NYS)
**Authority**: NYC Department of Buildings, NYS DOS

---

## Quick Reference - NYC vs. IBC

| Requirement | NYC Code | IBC 2021 |
|-------------|----------|----------|
| High-rise threshold | 75' (residential 75') | 75' |
| Sprinkler threshold (R-2) | All new | 3+ stories |
| Seismic design category | C | Varies |
| Snow load (NYC) | 30 psf | 30 psf |
| Wind speed (NYC) | 98-115 mph | 115-120 mph |
| Frost depth | 48" | 36" typical |
| Exit width factor (stairs) | 0.3" per occ | 0.3" per occ |
| Dead-end corridor max | 50' (sprinklered) | 50' (sprinklered) |

## Occupancy Classifications

### NYC Occupancy Group Differences

| Group | NYC | IBC Notes |
|-------|-----|-----------|
| J-1 | Residential (hotels) | R-1 in IBC |
| J-2 | Residential (apartments) | R-2 in IBC |
| J-3 | Residential (1-2 family) | R-3 in IBC |

### Assembly (Group A)
- **Place of assembly permit**: Required for 75+ persons
- **Certificate of occupancy**: Must list max occupant load
- **Place of assembly inspection**: Annual required

### High-Rise (FC 403)
NYC triggers:
- **Occupied floor > 75'** above fire department access
- **Class E buildings**: All high-rise office require enhanced systems

## Height and Area

### NYC Height Limits

| Group | Class IA | Class IB | Class II-A |
|-------|----------|----------|------------|
| A-1 | Unlimited | 12 stories | 2 stories |
| B | Unlimited | 12 stories | 4 stories |
| J-1 (R-1) | Unlimited | 12 stories | 5 stories |

### Zoning Height (NYC)
- **Determined by**: Floor Area Ratio (FAR) and zoning district
- **Sky exposure plane**: Per zoning resolution
- **Mechanical bulkhead**: Excluded from height
- **Rooftop structures**: Per zoning allowances

## Egress (NYC Chapter 10)

### Exit Width

#### Stairways
- **0.3" per occupant** without sprinklers
- **0.2" per occupant** with sprinklers
- **Minimum width**: 44" (36" if < 50 occupants)

#### Doors/Other Components
- **0.2" per occupant** without sprinklers
- **0.15" per occupant** with sprinklers

### Stair Requirements

#### Dimensional Requirements
| Element | NYC Requirement |
|---------|-----------------|
| Max riser | 7-3/4" (new), 8-1/4" (existing) |
| Min tread | 10" (new), 9" (existing) |
| Min headroom | 80" |
| Min width | 44" |
| Nosing | 3/4" - 1-1/4" projection |

#### Stair Enclosure
| Building Height | Rating |
|-----------------|--------|
| 4+ stories | 2 hours |
| < 4 stories | 1 hour |
| High-rise | Smokeproof or pressurized |

### Exit Signs (NYC Specific)
- **Power**: Battery backup 1.5 hours
- **Low-level exit signs**: Required in high-rise Group R
- **Photoluminescent path marking**: Required in high-rise buildings

### Accessible Means of Egress
- **Areas of refuge**: Required in non-sprinklered buildings
- **Elevators**: May be part of accessible egress (sprinklered)
- **Two-way communication**: Required at refuge areas

## Fire Protection

### Sprinkler Requirements (NYC)

#### Mandatory Sprinklers
- **All new J-2 (R-2)**: Required regardless of height
- **All high-rise buildings**: Required
- **Group A > 100 occupants**: Required
- **Group E**: Required throughout
- **Cellars and basements > 1,500 sf**: Required

#### Standpipe Requirements
| Building Type | Standpipe Class |
|---------------|-----------------|
| High-rise | Class I and III |
| 4+ stories | Class I |
| Below grade > 30' | Class I |
| Stage > 1,000 sf | Class III |

### Fire Alarm Requirements

#### When Required
- **Group A > 300 occupants**: Fire alarm
- **Group B > 500 occupants**: Fire alarm
- **Group E**: Fire alarm with voice
- **High-rise**: Voice alarm/evacuation
- **Group J-1/R-1**: Smoke detectors + notification

#### High-Rise Fire Safety
- **Fire safety director**: Required
- **Fire command station**: Required
- **Two-way communication**: Required in stairs
- **Elevator recall**: Required

## Accessibility (NYC)

### NYC Human Rights Law
- Goes beyond ADA in some areas
- Local Law 58 of 1987: Accessibility standards
- Prior-approved plans: May not meet current code

### Key NYC Accessibility Requirements

| Element | NYC Requirement | ADA |
|---------|-----------------|-----|
| Accessible entrance | At least 1 | At least 1 |
| Accessible units (new) | 5% Type A, 2% Type B | 5% |
| Signage | Room signs with Braille | Same |
| Toilet rooms | See below | See ADA |

### Accessible Dwelling Units (J-2)
- **Type A units**: 5% of total (2% in low-rise)
- **Type B units**: All ground floor (no elevator)
- **Type B units**: All units (elevator building)

## Structural (NYC)

### Seismic Design
- **Seismic Design Category**: C for most of NYC
- **Site class**: D assumed unless geotechnical report
- **Short period (Ss)**: 0.29g
- **1-second period (S1)**: 0.06g

### Wind Loads
- **Basic wind speed**: 98-115 mph (varies by borough)
- **Exposure category**: B typical (urban)
- **Hurricane provisions**: Not required (not hurricane-prone)

### Snow Loads
- **Ground snow load**: 30 psf
- **Flat roof snow load**: 30 psf minimum
- **Drift loads**: Per ASCE 7

### Foundation Requirements
- **Frost depth**: 48" minimum
- **Soil investigation**: Required for buildings > 3 stories
- **Rock anchors**: Common in Manhattan

---

## Mechanical (NYC)

### Ventilation
- Follows NYC Mechanical Code (based on IMC)
- Enhanced requirements for cooking operations
- Laboratory ventilation per NFPA 45

### Elevator Requirements

| Building Type | Elevator Required |
|---------------|-------------------|
| J-2 > 4 stories | Yes |
| J-2, 5+ units, any height | Yes (accessible) |
| B > 4 stories | Yes |
| All high-rise | Multiple, sized per code |

### Emergency Power
- **High-rise**: Required for elevators, fire pumps, emergency lights
- **Group I-2 (hospitals)**: Full emergency power
- **Generator location**: Per fire code

## Plumbing (NYC)

### Fixture Requirements
Similar to IPC with NYC amendments:
- **J-2 (apartments)**: 1 WC, 1 lav, 1 tub/shower per unit
- **Laundry**: 1 per 20 units or per unit

### Water Service
- **NYC DEP**: Issues water/sewer connections
- **Backflow**: RPZ required for many uses
- **Hot water**: 120Â°F max at fixtures

### Drainage
- **Combined sewers**: Still exist in much of NYC
- **Separation**: Required in some areas
- **Stormwater detention**: May be required (site-specific)

## NYC Local Laws

### Significant Local Laws

| Local Law | Year | Requirement |
|-----------|------|-------------|
| LL26 | 2004 | Sprinklers in existing high-rise residential |
| LL141 | 2013 | Facade inspection (FISP) every 5 years |
| LL152 | 2016 | Gas piping inspection |
| LL11 | 1998 | Facade inspection (buildings > 6 stories) |
| LL97 | 2019 | Carbon emissions limits |
| LL92/94 | 2019 | Green/solar roof requirements |

### Energy Code (NYC)
- **NYC Energy Conservation Code**: Based on IECC 2016+
- **NYC Stretch Energy Code**: More stringent
- **Local Law 97**: Carbon limits by 2024, 2030

### Facade Inspection (FISP)
- **Required for**: Buildings > 6 stories
- **Cycle**: Every 5 years
- **Report to DOB**: Within filing window
- **Categories**: Safe, SWARMP, Unsafe

## Quick Checklists

### NYC New Building Filing
- [ ] Zoning analysis complete
- [ ] Site plan shows all setbacks
- [ ] FAR calculation verified
- [ ] Building classification determined
- [ ] Exit analysis per occupant load
- [ ] Sprinkler/standpipe per code
- [ ] Accessibility (Type A, Type B units)
- [ ] Energy code compliance path
- [ ] Structural per NYC amendments
- [ ] DEP water/sewer approval

### NYC Alteration Filing
- [ ] Existing conditions documented
- [ ] Scope determines filing type (A1/A2/A3)
- [ ] Existing violations cleared
- [ ] Certificate of Occupancy reviewed
- [ ] Change of use triggers
- [ ] Accessibility upgrades required?
- [ ] FISP status confirmed
- [ ] Local Law 97 impact assessed

### High-Rise Requirements
- [ ] Smokeproof stairs or pressurization
- [ ] Class I & III standpipes
- [ ] Automatic sprinklers throughout
- [ ] Fire command center
- [ ] Voice alarm communication
- [ ] Two-way communication in stairs
- [ ] Elevator recall
- [ ] Emergency power (elevators, pumps, egress)
- [ ] Fire safety director designation
- [ ] Low-level exit signs (Group R)
