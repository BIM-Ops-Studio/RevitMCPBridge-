## Submittal Requirements

### Standard Submittals
| Type | When |
|------|------|
| Product data | Prior to fabrication |
| Shop drawings | Prior to fabrication |
| Samples | Prior to installation |
| Certificates | At closeout |

### Submittal Review
| Action | Meaning |
|--------|---------|
| Approved | Proceed |
| Approved as noted | Incorporate notes |
| Revise and resubmit | Correct and return |
| Rejected | Do not proceed |

## Quality Control

### Testing Requirements
| Test | Frequency |
|------|-----------|
| Concrete cylinders | Per ACI 301 |

### Inspection Points
| Phase | Inspections |
|-------|-------------|
| Foundation | Footing, slab, waterproofing |
| Structure | Steel, concrete, framing |
| Envelope | Air barrier, flashing, roofing |
| Interior | Framing, MEP rough-in |
