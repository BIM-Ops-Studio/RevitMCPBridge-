# Religious and Assembly Design

## Assembly Occupancy Types

### IBC Occupancy Classifications
| Use Group | Description | Example |
|-----------|-------------|---------|
| A-1 | Fixed seating, performing arts | Theater, concert hall |
| A-2 | Food/drink consumption | Restaurant, banquet hall |
| A-3 | Worship, recreation, without fixed seats | Churches, gymnasiums |
| A-4 | Indoor spectator sports | Arena, stadium |
| A-5 | Outdoor spectator areas | Bleachers, grandstands |

|-----|---------------|

## Acoustics for Worship

### Reverberation Time (RT60)
| Use | Target RT60 |
|-----|-------------|
| Contemporary worship | 0.8-1.2 sec |
| Traditional/liturgical | 1.5-2.5 sec |
| Organ music | 2.0-3.0 sec |
| Concert hall | 1.8-2.2 sec |
| Speech only | 0.6-0.8 sec |

### Background Noise
| Space | Maximum NC |
|-------|------------|
| Worship space | NC-25 to NC-30 |
| Chapel | NC-20 to NC-25 |
| Music rehearsal | NC-25 to NC-30 |
| Recording studio | NC-15 to NC-20 |

### Acoustic Treatments
| Element | Purpose |
|---------|---------|
| Absorptive ceiling | Control RT60 |
| Hard side walls | Sound reflection |
| Carpet (limited) | Mid-high absorption |
| Fabric seats | Absorption consistency |
| Diffusion panels | Even distribution |
| Bass traps | Low frequency control |

### Sound Reinforcement
| Congregation Size | System Type |
|-------------------|-------------|
| < 200 | Distributed ceiling, simple |
| 200-500 | Central cluster, delay |
| 500-1,000 | Line array, distributed |
| 1,000+ | Multiple clusters, extensive delays |

## HVAC Considerations

### Load Characteristics
| Condition | Consideration |
|-----------|---------------|
| High occupancy | Significant cooling load |
| Variable attendance | Part-load operation |
| Tall ceilings | Stratification |
| Large volume | Long warm-up |

| Standard | Rate |
|----------|------|
| ASHRAE 62.1 | 5 CFM/person + 0.06 CFM/SF |
| Minimum | 15 CFM/person |
| Recommended | 20 CFM/person |

| System | Application |
|--------|-------------|
| Underfloor | Theater seating |
| Overhead | Flexible spaces |
| Displacement | Large volumes |
| Radiant | Historic renovation |

### Noise Control
