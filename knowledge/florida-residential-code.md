# Florida Residential Code

**Based on**: 7th Edition Florida Building Code - Residential (2023)
**Base Code**: IRC 2021 with Florida amendments
**Scope**: One/two-family dwellings, townhouses (≤3 stories)
**Authority**: Florida Building Commission, Local Building Officials

---

## Scope and Application

### What's Covered (R101.2)
- **Detached one- and two-family dwellings**
- **Townhouses**: Max 3 stories, separate means of egress
- **Accessory structures**: Garages, carports, sheds

### What's NOT Covered
- **3+ family**: Use Florida Building Code (R-2)
- **4+ stories**: Use Florida Building Code
- **Mixed-use**: Use Florida Building Code
- **Hotels/motels**: Use Florida Building Code (R-1)

## Building Planning (Chapter 3)

### Room Dimensions (R304)

#### Minimum Areas
| Room Type | Minimum |
|-----------|---------|
| Habitable room | 70 sf |
| At least 1 habitable room | 120 sf |
| Kitchen | No minimum (functional layout) |
| Bathroom | No minimum (fixture clearances) |

#### Minimum Width
| Room Type | Minimum Width |
|-----------|---------------|
| Habitable room | 7'-0" |
| Hallway | 3'-0" |
| Bathroom | Per fixture clearances |

### Ceiling Height (R305)

| Space | Minimum Height |
|-------|----------------|
| Habitable rooms | 7'-0" |
| Bathrooms | 7'-0" (or 6'-8" if sloped) |
| Hallways | 7'-0" |
| Basements (habitable) | 7'-0" |
| Basements (non-habitable) | 6'-8" |
| Beams/girders (min clearance) | 6'-4" |

**Sloped ceilings**: At least 50% of room at full height

### Light and Ventilation (R303)

#### Natural Light
- **Glazing area**: 8% of floor area minimum
- **Alternative**: Artificial light per code

#### Natural Ventilation
- **Opening area**: 4% of floor area minimum
- **Alternative**: Mechanical ventilation

#### Mechanical Ventilation (Required)
| Space | Rate |
|-------|------|
| Bathroom (no window) | 50 CFM intermittent / 20 CFM continuous |
| Kitchen | 100 CFM intermittent / 25 CFM continuous |
| Whole house | 0.35 ACH or per ASHRAE 62.2 |

### Emergency Escape (R310)

#### Egress Windows Required
- **Every sleeping room**: At least 1 egress window
- **Basements with habitable space**: At least 1 egress window

#### Egress Window Dimensions
| Requirement | Value |
|-------------|-------|
| Minimum opening area | 5.7 sf |
| Minimum opening height | 24" |
| Minimum opening width | 20" |
| Maximum sill height | 44" AFF |
| Grade floor/basement | 5.0 sf opening OK |

#### Window Wells
- **Minimum area**: 9 sf
- **Minimum dimension**: 36" (horizontal)
- **Ladder/steps**: If > 44" deep

## Foundations (Chapter 4)

### Florida Foundation Requirements

#### Frost Protection
- **Frost depth**: N/A (no frost in Florida)
- **Minimum footing depth**: 12" below undisturbed soil

#### Foundation Types
| Type | Florida Use |
|------|-------------|
| Slab-on-grade | Most common |
| Stem wall | Elevated (flood zones) |
| Piles/piers | Coastal, flood zones |
| Crawlspace | Limited (moisture issues) |

### Slab-on-Grade (R506)

| Element | Requirement |
|---------|-------------|
| Minimum thickness | 3.5" |
| Vapor retarder | 6 mil polyethylene |
| Base course | 4" gravel/sand |
| Reinforcement | Per design (WWM typical) |
| Control joints | 15' max spacing |

### Footings (R403)

| Footing Type | Minimum Dimensions |
|--------------|-------------------|
| 1-story (frame) | 12" W × 6" D |
| 2-story (frame) | 15" W × 6" D |
| 1-story (masonry) | 12" W × 6" D |
| 2-story (masonry) | 15" W × 7" D |
| 3-story | Per engineering |

### Anchor Bolts (R403.1.6)

| Wind Speed | Bolt Size | Spacing |
|------------|-----------|---------|
| < 130 mph | 1/2" | 6' o.c. |
| 130-150 mph | 1/2" | 4' o.c. |
| > 150 mph | 5/8" | 4' o.c. |
| HVHZ | 5/8" | 32" o.c. max |

**Location**: Within 12" of corners, 12" of openings

## Wall Covering (Chapter 7)

### Exterior Wall Covering

#### Florida Requirements
- **Product approval**: Required
- **Water-resistive barrier**: Required behind cladding
- **Flashing**: At penetrations, transitions

#### Stucco (R703.7)
- **Thickness**: 7/8" minimum (3-coat)
- **Lath**: Galvanized or stainless
- **WRB**: 2 layers Grade D paper or equivalent
- **Control joints**: 144 sf max, 18' max

#### Vinyl Siding (R703.11)
- **Wind rating**: Per design wind speed
- **Florida Product Approval**: Required
- **Underlayment**: WRB required

#### Masonry Veneer (R703.8)
- **Ties**: 1 per 2.67 sf (corrugated), 1 per 3.5 sf (adjustable)
- **Air space**: 1" minimum
- **Weeps**: At base, above openings
- **Flashing**: Required

### Interior Wall Covering

#### Gypsum Board (R702)
| Application | Minimum Thickness |
|-------------|-------------------|
| Walls | 1/2" |
| Ceilings (16" o.c.) | 1/2" |
| Ceilings (24" o.c.) | 5/8" |
| Water-resistant (wet areas) | 1/2" |

## Chimneys and Fireplaces (Chapter 10)

### Masonry Fireplaces (R1001)
- **Firebox depth**: 20" minimum
- **Hearth extension**: 16" front, 8" sides
- **Clearance to combustibles**: 2" (masonry), 6" (metal)

### Factory-Built Fireplaces (R1004)
- **Listing**: UL 127
- **Installation**: Per manufacturer
- **Clearances**: Per listing

### Chimneys
- **Height**: 3' above roof, 2' above anything within 10'
- **Spark arrestor**: Required (wildfire areas)

## Florida-Specific Requirements

### Hurricane/Wind
- Enhanced nailing schedules
- Roof-to-wall connectors
- Impact glazing or shutters
- Product approval required

### Termite Protection
- Soil treatment or bait system
- Pressure-treated wood at ground
- Inspection access

### Flood Zones
- Elevation per FEMA/local
- V-zone: Pile construction
- A-zone: Elevation or dry floodproofing

### Energy
- Florida Energy Code compliance
- HERS rating available
- Solar-ready provisions
