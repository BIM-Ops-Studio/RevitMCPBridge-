## Color and Finishes

### Color Temperature
| Color Type | Temperature | Effect |
|------------|-------------|--------|
| Warm (red, orange, yellow) | Advancing | Cozy, intimate |
| Cool (blue, green, purple) | Receding | Calm, spacious |
| Neutral (white, gray, beige) | Variable | Versatile, timeless |

### Color Schemes
| Scheme | Description | Use |
|--------|-------------|-----|
| Monochromatic | Single hue, varied values | Sophisticated |
| Analogous | Adjacent hues | Harmonious |
| Complementary | Opposite hues | Dynamic |
| Split-complementary | One hue + two adjacents | Balanced |
| Triadic | Three equidistant hues | Vibrant |

### 60-30-10 Rule
| Proportion | Application |
|------------|-------------|
| 60% | Dominant (walls, large surfaces) |
| 30% | Secondary (upholstery, curtains) |
| 10% | Accent (accessories, art) |

### Light Reflectance Values (LRV)
| Surface | Recommended LRV |
|---------|-----------------|
| Ceiling | 80-90% |
| Walls | 50-70% |
| Floors | 20-40% |
| Furniture | 25-45% |

## Acoustic Considerations

|----------|-----------|
| Drywall | 0.05 |
| Carpet | 0.25-0.55 |
| Acoustic tile | 0.50-0.90 |
| Fabric panels | 0.60-0.85 |
| Curtains | 0.35-0.55 |

### Room Treatments
| Issue | Solution |
|-------|----------|
| Echo | Add soft surfaces |
| Noise transfer | Mass, sealing |
| HVAC noise | Duct lining, silencers |
| Impact noise | Underlayment, carpet |
