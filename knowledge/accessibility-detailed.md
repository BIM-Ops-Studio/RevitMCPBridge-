# Accessibility Design (ADA/FHA)

## Accessible Routes

### Clear Width
| Element | Minimum |
|---------|---------|
| Walking surface | 36" |
| Passing space | 60" × 60" (every 200') |
| U-turn around obstruction | 42" min each side |
| Doorway | 32" clear |
| Corridor | 36" (44" preferred) |

### Clear Floor Space
| Type | Dimensions |
|------|------------|
| Forward approach | 30" × 48" |
| Parallel approach | 30" × 48" |
| Wheelchair turning | 60" diameter circle |
| T-shaped turning | 60" × 60" T |

### Protruding Objects
| Condition | Requirement |
|-----------|-------------|
| Wall-mounted (27"-80" AFF) | 4" max projection |
| Post-mounted (27"-80" AFF) | 12" max projection |
| Objects below 27" | Detectable (cane) |
| Overhead clearance | 80" minimum |

### Ground and Floor Surfaces
| Feature | Requirement |
|---------|-------------|
| Slip resistance | Stable, firm |
| Carpet pile | 1/2" max |
| Carpet pad | Firm |
| Changes in level | 1/4" max vertical |
| Beveled changes | 1/4"-1/2" at 1:2 slope |
| Gratings | 1/2" max openings |

## Ramps

### Slope Requirements
| Ramp Type | Maximum Slope |
|-----------|---------------|
| New construction | 1:12 (8.33%) |
| Existing (rise < 3") | 1:8 |
| Existing (rise 3"-6") | 1:10 |
| Cross slope | 1:48 (2%) max |
| Curb ramp | 1:12 |

### Dimensions
| Feature | Requirement |
|---------|-------------|
| Width | 36" clear minimum |
| Rise per run | 30" maximum |
| Landing length | 60" minimum |
| Landing width | At least ramp width |

### Handrails
| Feature | Requirement |
|---------|-------------|
| Required when | Rise > 6" |
| Height | 34"-38" |
| Clearance from wall | 1-1/2" |
| Top extension | 12" horizontal |
| Bottom extension | 12" + ramp slope |
| Graspability | 1-1/4" to 2" diameter |

### Edge Protection
| Type | Requirement |
|------|-------------|
| Curb | 4" minimum height |
| Rail | Extended to floor |
| Wall | At ramp edge |
| Required when | Drop-off > 4" |

## Fair Housing Act

### Covered Dwellings
| Building Type | Requirements |
|---------------|--------------|
| 4+ units, elevator | All units |
| 4+ units, no elevator | Ground floor only |
| First occupancy | After March 13, 1991 |

### Seven Design Requirements
1. Accessible building entrance
2. Accessible common areas
3. Usable doors (32" clear)
4. Accessible route into/through unit
5. Light switches, outlets at accessible heights
6. Reinforced bathroom walls for grab bars
7. Usable kitchens and bathrooms

### Kitchen/Bath Clear Floor
| Layout | Clearance |
|--------|-----------|
| U-shaped kitchen | 60" between counters |
| Galley kitchen | 40" between counters |
| Bathroom | 30" × 48" at fixtures |
