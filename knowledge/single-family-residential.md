# Single-Family Residential Design

## House Types

### By Style
| Style | Characteristics |
|-------|-----------------|
| Ranch | Single story, spread layout, attached garage |
| Two-story | Bedrooms up, living down, efficient footprint |
| Split-level | Staggered floors, grade changes |
| Cape Cod | 1.5 story, dormers, steep roof |
| Colonial | Symmetrical, center entry, formal |
| Contemporary | Open plan, large glass, flat/shed roofs |
| Mediterranean | Stucco, tile roof, courtyards |
| Craftsman | Low roof, exposed rafters, porches |
| Farmhouse | Gabled roof, porch, simple forms |
| Modern | Minimal, geometric, indoor-outdoor |

## Florida-Specific Requirements

### Hurricane Protection
| Element | Requirement |
|---------|-------------|
| Windows/doors | Impact-rated or shutters |
| Garage door | Wind-rated |
| Roof-to-wall | Hurricane straps |
| Roof sheathing | Enhanced nailing |
| Entry doors | Impact-rated |

### Flood Zone
| Zone | Requirement |
|------|-------------|
| AE, VE | Elevate to BFE + freeboard |
| X shaded | Consider elevation |
| Enclosures below BFE | Breakaway walls, flood vents |
