# Fire Protection Systems

## Sprinkler Systems

### System Types
| Type | Description | Use |
|------|-------------|-----|
| Wet pipe | Water in pipes, immediate discharge | Most common, heated spaces |
| Dry pipe | Air in pipes, water held at valve | Unheated spaces, freezing risk |
| Pre-action | Dry + detection before water | Data centers, museums |
| Deluge | Open heads, all discharge at once | High hazard, aircraft hangars |

### Hazard Classifications
| Class | Examples | Density |
|-------|----------|---------|
| Light | Office, church, hotel | 0.10 gpm/SF |
| Ordinary Group 1 | Parking, restaurant | 0.15 gpm/SF |
| Ordinary Group 2 | Retail, warehouse (light) | 0.20 gpm/SF |
| Extra Hazard Group 1 | Printing, woodworking | 0.30 gpm/SF |
| Extra Hazard Group 2 | Flammable liquids | 0.40 gpm/SF |

### Coverage Areas
| Hazard | Max Area/Head | Max Spacing |
|--------|---------------|-------------|
| Light | 225 SF | 15' × 15' |
| Ordinary | 130 SF | 15' × 15' |
| Extra | 90-100 SF | 12' × 12' |

### Head Types
| Type | Orientation | Use |
|------|-------------|-----|
| Pendent | Down from ceiling | Standard concealed ceiling |
| Upright | Up from pipe | Exposed, warehouse |
| Sidewall | Horizontal from wall | Corridors, small rooms |
| Concealed | Flush with ceiling | Architectural finish |
| ESFR | Early suppression | High-pile storage |

### Sprinkler Clearances
| Clearance | Requirement |
|-----------|-------------|
| Below ceiling | 1" to 12" (standard pendent) |
| Below sprinkler | 18" clear to storage |
| To walls | 4" min from sidewall |
| To obstructions | Per NFPA 13 rules |

## Standpipes

### Classes
| Class | Use | Outlets |
|-------|-----|---------|
| Class I | Fire dept use | 2-1/2" hose connection |
| Class II | Occupant use | 1-1/2" hose cabinet |
| Class III | Both | 2-1/2" + 1-1/2" reducer |

### When Required
- Buildings > 3 stories
- Buildings with floor > 30' above/below fire dept access
- Covered malls
- Stages > 1,000 SF
- Underground buildings

### Standpipe Locations
- In enclosed stairs
- Within 200' travel to all portions of floor
- At roof level (Class I)
- Horizontal standpipes for large floor plates

## Fire Alarm Systems

### System Types
| Type | Description | Use |
|------|-------------|-----|
| Conventional | Zone-based, non-addressable | Small buildings |
| Addressable | Device-specific identification | Most commercial |
| Analog addressable | Variable sensitivity | Large/complex buildings |
| Voice evacuation | Speakers for announcements | High-rise, assembly |

### Initiating Devices
| Device | Spacing | Use |
|--------|---------|-----|
| Smoke detector | 30' radius (900 SF) | Corridors, offices |
| Heat detector | 50' spacing | Kitchens, garages |
| Duct detector | In HVAC ducts > 2,000 CFM | Air handlers |
| Manual pull station | At exits | All buildings |
| Sprinkler waterflow | On risers | Sprinklered buildings |

### Notification Appliances
| Device | Spacing | Notes |
|--------|---------|-------|
| Horn/strobe | Per room coverage | 75 dBA min |
| Visual (strobe) | 50' max spacing | ADA required |
| Speaker | Voice evacuation | High-rise, large assembly |

### Fire Alarm Requirements
| Building Type | Requirement |
|---------------|-------------|
| Assembly > 300 occ | Manual + auto |
| Business > 3 stories | Manual + auto (sprinklered exempt) |
| Educational | Manual + auto |
| High-rise (> 75') | Voice evacuation |
| Healthcare | Full addressable |

## Fire-Rated Construction

### Fire Barrier vs Fire Partition
| Type | Rating | Continuity | Use |
|------|--------|------------|-----|
| Fire barrier | 1-4 hr | Deck to deck | Occupancy separation |
| Fire partition | 1 hr | Deck to deck | Corridor walls |
| Smoke barrier | 1 hr | Deck to deck | Smoke compartments |
| Smoke partition | 0 hr | Deck to deck (may have openings) | Smoke control |

### Required Fire Ratings
| Separation | Rating Required |
|------------|-----------------|
| Dwelling unit separation | 1 hour |
| Corridor walls (R-2, I) | 1 hour |
| Corridor walls (B, M) | 0-1 hour |
| Exit enclosure (< 4 stories) | 1 hour |
| Exit enclosure (≥ 4 stories) | 2 hours |
| Occupancy separation | Per IBC Table 508.4 |
| Shaft enclosure | 1-2 hours |

### Fire Door Ratings
| Wall Rating | Door Rating |
|-------------|-------------|
| 3-4 hour | 3 hour (A label) |
| 2 hour | 1-1/2 hour (B label) |
| 1 hour | 1 hour (B) or 3/4 hour (C) |
| 1 hour (corridor) | 20 minute (C) |
| Smoke barrier | 20 minute + S label |

### Opening Protection (Fire Windows)
| Wall Rating | Max Opening | Window Rating |
|-------------|-------------|---------------|
| 1 hour | 25% of wall | 3/4 hour |
| 2 hour | Limited | Not typically allowed |
| Sprinklered | May increase allowable | Per code |

## Means of Egress

### Travel Distance Limits
| Occupancy | Unsprinklered | Sprinklered |
|-----------|---------------|-------------|
| Assembly | 200' | 250' |
| Business | 200' | 300' |
| Educational | 200' | 250' |
| Residential | 200' | 250' |
| Storage (S-1) | 200' | 400' |
| Hazardous | 75' | 100' |

### Common Path of Travel
| Occupancy | Limit |
|-----------|-------|
| Assembly (≤ 30 occ) | 75' |
| Business | 100' |
| Industrial | 100' |
| Storage | 100' |
| Others | 75' |

### Dead End Corridors
- 20' max (unsprinklered)
- 50' max (sprinklered)
- Exception: some occupancies allow more

### Exit Separation
| Number of Exits | Separation |
|-----------------|------------|
| 2 exits | 1/2 diagonal |
| 3+ exits | 1/3 diagonal (at least 2) |

## Smoke Control

### When Required
- Covered malls
- Atriums
- Underground buildings
- High-rise buildings (some jurisdictions)
- Stages over 50' height

### Methods
| Method | Description |
|--------|-------------|
| Smoke exhaust | Fans remove smoke |
| Pressurization | Positive pressure in protected spaces |
| Smoke barriers | Limit spread within floor |
| Opening protection | Automatic closing doors |

### Atrium Smoke Control
- Mechanical exhaust sized for fire size
- Minimum 6 air changes/hour
- Makeup air required
- Acceptance testing required

## Fire Department Access

### Fire Lane Requirements
| Element | Requirement |
|---------|-------------|
| Width | 20' min (26' with aerial) |
| Vertical clearance | 13'-6" min |
| Turning radius | 25' inside, 50' outside |
| Surface | All-weather, support 75,000 lbs |

### Fire Hydrant Spacing
| Use | Max Spacing |
|-----|-------------|
| Residential | 500' |
| Commercial | 300' |
| Industrial | 300' |

### FDC (Fire Department Connection)
- Within 100' of fire hydrant
- Visible and accessible
- Signage required
- One per system/zone

## Firestopping

### Penetration Types
| Type | Treatment |
|------|-----------|
| Through penetration | Full seal both sides |
| Membrane penetration | Seal at membrane |
| Joint | Fire-rated sealant |
| Curtain wall perimeter | Safing insulation + sealant |

### Firestop Materials
| Material | Use |
|----------|-----|
| Intumescent | Pipe, cable penetrations |
| Mineral fiber | Large openings |
| Caulk/sealant | Joints, small openings |
| Pillows | Temporary, removable |
| Cast-in device | Pipes, sleeves |

### Documentation
- UL or FM listed systems
- Match assembly to actual condition
- Inspection and sign-off
- Maintain log of all penetrations
