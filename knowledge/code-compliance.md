# Building Code Compliance - Quick Reference

## Egress Requirements (IBC)

### Minimum Door Widths
| Occupancy | Minimum Clear Width |
|-----------|---------------------|
| Residential (single unit) | 32" clear |
| Residential (common areas) | 36" clear |
| Commercial / Assembly | 36" clear |
| Accessible routes | 32" clear minimum |

### Minimum Corridor Widths
| Use | Minimum Width |
|-----|---------------|
| Residential | 36" |
| Commercial (occupant load < 50) | 44" |
| Commercial (occupant load ≥ 50) | 44" |
| Healthcare | 96" (8'-0") |
| Educational | 72" (6'-0") |

### Stair Requirements
| Element | Residential | Commercial |
|---------|-------------|------------|
| Minimum width | 36" | 44" |
| Maximum riser | 7-3/4" | 7" |
| Minimum tread | 10" | 11" |
| Headroom | 6'-8" | 6'-8" |
| Handrail height | 34" - 38" | 34" - 38" |
| Guard height | 36" | 42" |

### Number of Exits Required
| Occupant Load | Minimum Exits |
|---------------|---------------|
| 1-500 | 2 |
| 501-1000 | 3 |
| Over 1000 | 4 |

## Accessibility (ADA/ABA/FBC)

### Accessible Route Requirements
- Minimum 36" wide
- 60" x 60" turning space at changes in direction
- Maximum 1:20 slope (5%) for walking surfaces
- Maximum 1:12 slope (8.33%) for ramps

### Door Requirements (Accessible)
- 32" minimum clear width
- Maximum 5 lbs operating force (interior)
- Maximum 8.5 lbs operating force (exterior/fire)
- 18" minimum clearance on pull side at latch
- Thresholds maximum 1/2" (1/4" for sliding)

### Restroom Requirements (Accessible)
- 60" turning diameter
- Toilet 17"-19" seat height
- Grab bars: 42" side, 36" rear
- Lavatory knee clearance: 27" AFF minimum
- Mirror: 40" maximum to bottom of reflective surface

### Parking Requirements
| Total Spaces | Accessible Required |
|--------------|---------------------|
| 1-25 | 1 |
| 26-50 | 2 |
| 51-75 | 3 |
| 76-100 | 4 |
| 101-150 | 5 |
| 151-200 | 6 |

## Fire Ratings

### Fire-Rated Assemblies
| Assembly | Typical Rating |
|----------|----------------|
| Corridor walls (commercial) | 1-hour |
| Stair enclosure (3 stories or less) | 1-hour |
| Stair enclosure (4+ stories) | 2-hour |
| Shaft walls | 1-hour or 2-hour |
| Occupancy separation (most) | 1-hour |
| Occupancy separation (H occupancy) | 2-hour |

### Door Ratings in Rated Walls
| Wall Rating | Door Rating |
|-------------|-------------|
| 1-hour | 45-minute (3/4 hour) |
| 2-hour | 90-minute (1-1/2 hour) |
| 3-hour | 90-minute (1-1/2 hour) |

### Fire-Rated Door Labels
When reviewing doors, check for:
- Fire rating shown in door schedule
- Doors in rated corridors are rated
- Stair doors are rated appropriately
- Labels match wall ratings (per table above)

## Plumbing Fixture Counts (IPC)

### Residential (per dwelling unit)
- 1 water closet
- 1 lavatory
- 1 bathtub or shower
- 1 kitchen sink
- 1 laundry facility (washer connection)

### Commercial (Assembly)
| Occupant Load | Water Closets | Lavatories |
|---------------|---------------|------------|
| 1-50 | 1M / 1F | 1 |
| 51-100 | 2M / 2F | 2 |
| 101-200 | 3M / 3F | 2 |
| 201-400 | 4M / 4F | 3 |

## QC Checklist - Code Items

When reviewing drawings, verify:
- [ ] All egress paths have required width
- [ ] Door swings do not block egress
- [ ] Stair dimensions meet code
- [ ] Required number of exits provided
- [ ] Exit separation distance met (1/2 diagonal)
- [ ] Accessible route from parking to entrance
- [ ] Accessible restrooms provided
- [ ] Fire ratings shown where required
- [ ] Rated doors in rated walls
- [ ] Fixture counts meet occupancy requirements
