## Fire Protection Systems

### Automatic Sprinkler Systems

#### NFPA 13 - Standard Occupancies

| System Type | Application |
|-------------|-------------|
| Wet pipe | Most common, heated spaces |
| Dry pipe | Unheated spaces, freezing risk |
| Pre-action | High-value, water-sensitive |
| Deluge | High-hazard, rapid spread |

#### NFPA 13R - Residential (up to 4 stories)

| Feature | Requirement |
|---------|-------------|
| Coverage | Dwelling units, corridors, common areas |
| Omit | Attics, closets <24 sf, bathrooms <55 sf |
| Water supply | Domestic acceptable |

#### NFPA 13D - One/Two-Family

| Feature | Requirement |
|---------|-------------|
| Coverage | All areas (exceptions allowed) |
| Omit | Attics, garages, small bathrooms/closets |
| Water supply | 10-minute demand |
| Pipe | CPVC or PEX acceptable |

### Sprinkler Design Criteria

| Occupancy | Density (gpm/sf) | Area (sf) |
|-----------|------------------|-----------|
| Light hazard | 0.10 | 1,500 |
| Ordinary Group 1 | 0.15 | 1,500 |
| Ordinary Group 2 | 0.20 | 1,500 |
| Extra Hazard Group 1 | 0.30 | 2,500 |
| Extra Hazard Group 2 | 0.40 | 2,500 |

### Quick-Response Sprinklers
Required in:
- Light hazard occupancies
- Residential occupancies
- Healthcare patient areas
- Hotels/motels

## Standpipe Systems (NFPA 14)

### System Classes

| Class | Hose Connection | Use |
|-------|-----------------|-----|
| I | 2-1/2" | Fire department only |
| II | 1-1/2" | Building occupants |
| III | 2-1/2" + 1-1/2" | Both |

### When Required

| Building Type | Class Required |
|---------------|----------------|
| 4+ stories | Class I |
| High-rise (>75') | Class I |
| Covered mall | Class I |
| Stage >1,000 sf | Class III |
| Underground (>30' below) | Class I |

### Standpipe Location
- **Within 200'** of any point via travel path
- **At each floor landing** of required stairs
- **Roof access** in high-rise

## Fire Alarm Systems (NFPA 72)

### System Types

| Type | Description | Use |
|------|-------------|-----|
| Manual | Pull stations only | Small occupancies |
| Automatic | Detection + manual | Most occupancies |
| Voice | Voice evacuation | High-rise, assembly |
| Mass notification | Integrated alerts | Large facilities |

### When Required

| Condition | Alarm Required |
|-----------|----------------|
| Group A > 300 occupants | Yes |
| Group B > 500 occupants | Yes |
| Group E | Yes |
| Group H | Yes |
| Group I | Yes |
| Group R-1 | Yes |
| Group R-2 (3+ stories) | Yes |
| High-rise | Yes (voice) |

### Detection Requirements

| Location | Detection Type |
|----------|----------------|
| Sleeping areas | Smoke (single/multi-station) |
| Common areas | Smoke or heat |
| Mechanical rooms | Heat |
| Cooking areas | Heat or listed smoke |
| Storage | Per hazard |

### Notification Appliances

| Occupancy | Audible | Visual |
|-----------|---------|--------|
| All occupancies | Yes | Yes (public areas) |
| Sleeping areas | 75 dBA | Yes (ADA) |
| Assembly | 15 dBA above ambient | Yes |
| High-rise | Voice | Yes |

## Means of Egress Maintenance

### Exit Access
- **Corridors**: Kept clear at all times
- **Doors**: Operable, unobstructed
- **Aisles**: Minimum width maintained
- **Stairways**: Free of storage

### Exit Doors
| Element | Requirement |
|---------|-------------|
| Hardware | Operable from inside |
| Locks | No double-keying |
| Panic hardware | Maintained, tested |
| Closers | Adjusted, functional |
| Signs | Illuminated, visible |

### Emergency Lighting
- **Testing**: Monthly (30-second), annual (90-minute)
- **Documentation**: Written records
- **Maintenance**: Replace failed units promptly

### Exit Signs
- **Illumination**: Always on or upon power failure
- **Battery backup**: Tested monthly
- **Visibility**: Unobstructed

## Fire Extinguishers

### Placement Requirements

| Hazard Class | Max Travel | Size |
|--------------|------------|------|
| Class A (light) | 75' | 2A |
| Class A (ordinary) | 75' | 2A |
| Class B (light) | 50' | 10B |
| Class B (ordinary) | 50' | 20B |
| Class C | Based on A or B | - |
| Class K | 30' | Listed |

### Mounting Height
- **< 40 lbs**: Handle ≤ 5' AFF
- **> 40 lbs**: Handle ≤ 3.5' AFF
- **Above floor**: Minimum 4" AFF

### Inspection/Maintenance
| Interval | Activity |
|----------|----------|
| Monthly | Visual inspection |
| Annual | Professional maintenance |
| 6 years | Internal exam (stored pressure) |
| 12 years | Hydrostatic test |

## Hazardous Materials

### Storage Requirements

#### Flammable/Combustible Liquids

| Quantity | Storage Requirements |
|----------|---------------------|
| < 5 gallons | Approved container |
| 5-25 gallons | Safety can or cabinet |
| > 25 gallons | Approved storage room |
| Bulk | Approved tank |

#### Flammable Storage Cabinets
- **Listed**: FM or UL approved
- **Capacity**: Max 60 gallons Class I/II, 120 gallons Class III
- **Ventilation**: Not required but recommended
- **Self-closing**: Required

### Compressed Gases

| Gas Type | Indoor Limit |
|----------|--------------|
| Flammable | Per fire code table |
| Oxidizer | Per fire code table |
| Toxic | Per fire code table |
| Cryogenic | Per fire code table |

### Control Areas
- Limit quantities per control area
- Number of control areas per floor
- Required fire barriers

## Hot Work Operations

### Permit Required
- Welding
- Cutting
- Brazing
- Soldering
- Open flame

### Safety Requirements

| Element | Requirement |
|---------|-------------|
| Clear area | 35' of combustibles |
| Fire watch | During + 60 min after |
| Fire extinguisher | Within 10' |
| Notification | Fire alarm may need bypass |

### Prohibited Locations
- Sprinklered areas with impaired sprinklers
- Near flammable storage
- Explosive atmospheres
- Unauthorized areas

## Assembly Occupancies

### Occupant Load Posting
- Required for assembly spaces
- Posted at main entrance
- Per fire marshal determination

### Crowd Management
| Occupant Load | Requirement |
|---------------|-------------|
| > 250 | Crowd manager required |
| > 1,000 | Multiple crowd managers |
| Special events | Fire watch may be required |

### Egress Maintenance
- Maintain aisles per code
- No blocked exits
- Emergency lighting functional
- Exit signs visible

### Special Events
- Permit required for tents, canopies
- Cooking operations reviewed
- Pyrotechnics require permit
- Fire watch as required

## Healthcare Facilities

### Smoke Compartments
- Max 22,500 sf
- Max 200' travel within compartment
- Smoke barriers to underside of floor above

### Defend in Place
- Not immediate evacuation
- Move patients horizontally
- Smoke barriers critical

### Fire Drills
| Drill Type | Frequency |
|------------|-----------|
| Fire drill | Quarterly |
| Staff training | Per OSHA |
| Documentation | Required |

### Corridor Requirements
- 8' width (bed movement)
- No storage in corridors
- Smoke-resistant walls
- Self-closing doors

## Educational Facilities (Schools)

### Fire Drills (Florida)
| Drill Type | Frequency |
|------------|-----------|
| Fire drill | Monthly |
| Severe weather | Annually |
| Lockdown | Annually |

### Portable Heaters
- Generally prohibited
- Exception by fire marshal approval

### Decorations
- Limited combustible decorations
- No decorations on exits
- Treated materials or inherently non-combustible

## Hotels and Motels

### Smoke Alarms
- Each sleeping room
- Hardwired with battery backup
- Interconnected or connected to alarm

### Notification
- Audible in sleeping rooms
- Visual in accessible rooms
- Voice evacuation (high-rise)

### Fire Drills
- Staff training required
- Documentation required
- Guest notification procedures

### Egress
- Egress plan in each room
- Exit signs visible
- Emergency lighting functional

## Fire Department Access

### Fire Lanes
- 20' minimum width (26' aerial)
- Posted "No Parking Fire Lane"
- Maintained clear 24/7

### Fire Hydrants
- Per local requirements
- Accessible, unobstructed
- Maintained by utility/owner

### Knox Box/Key Box
- Fire department access
- Master keyed system
- Contains access keys
- Alarm codes (if required)

### Aerial Access
- Building location
- Setback requirements
- Overhead clearance

## Fire Watch

### When Required
- Impaired fire protection systems
- Hot work operations
- Special events
- Construction in occupied buildings

### Requirements
- Trained personnel
- Continuous patrol
- Communication equipment
- Fire extinguisher
- Knowledge of building

### Documentation
- Log entries every 15-30 minutes
- Report any hazards
- Maintain until system restored
