## Sheet Organization

### Numbering System
```
[Discipline][Type].[Sequence]

Examples:
A0.1 = Cover Sheet
A1.1 = Abbreviations, Notes & Symbols
A2.1 = Proposed Ground Floor Plan
A2.2 = Proposed Second Floor Plan
A3.1 = Building Elevations
A4.1 = Building Sections
A5.1 = Door/Window Schedule
```

### Sheet Types
| Number | Content |
|--------|---------|
| X0.X | Cover, General Info |
| X1.X | Notes, Symbols, Abbreviations |
| X2.X | Floor Plans, RCP, Roof Plan |
| X3.X | Elevations |
| X4.X | Sections |
| X5.X | Schedules |
| X6.X | Interior Elevations |
| X7.X | Enlarged Plans & Details |
| X8.X | Typical Details |

### Sheet Naming Convention
- Use "PROPOSED" prefix for new work views
- Examples:
  - PROPOSED GROUND FLOOR PLAN
  - PROPOSED SECOND FLOOR PLAN
  - PROPOSED BUILDING ELEVATIONS - NORTH & EAST
  - PROPOSED ROOF PLAN

## Keynotes

### Format
- Symbol: **Hexagon** (FC_Keynote_Hexagon family)
- Numbering: Two-digit format (01, 02, 03...)
- With description text adjacent to symbol

### Content Style
- NO "NEW" prefix (new construction assumed)
- Include contractor responsibility (SHALL language)
- Include MEP/structural coordination requirements
- Reference related drawings and schedules
- Use standard abbreviations (UNO, O.C., TYP.)

### Keynote Placement Rules
- Place near element being referenced
- Use leader to point to specific element
- Leader angle: 30, 45, or 60 degrees
- Do not overlap other annotations
- Equal spacing in legends (0.09 feet / ~1 inch)

## Annotations

### Text Heights
| Use | Height |
|-----|--------|
| Room names | 3/16" |
| Notes | 3/32" |
| Titles | 1/4" |
| Sheet titles | 3/8" |

### Leader Style
- Type: Straight with single bend
- Angle: 30, 45, or 60 degrees preferred
- Arrow points TO element

### Tags
- Door tags: Per door schedule
- Window tags: Per window schedule
- Room tags: Name and number

## Florida-Specific Requirements

### Impact Ratings
- All windows: Impact-rated per Florida Building Code
- Exterior doors: Impact-rated
- Garage doors: Wind-rated

### Construction Notes
- Reference FBC (Florida Building Code)
- Include wind load requirements where applicable
- Note hurricane protection requirements

---

## Specific Conventions

### Legend Organization
- SITE PLAN KEY NOTES - for site plan keynotes (SP-##)
- PLAN KEYNOTES - for floor plan keynotes (FP-##)
- Equal spacing between entries
- Border around legend
- Title at top

### View Naming
- Use "PROPOSED" prefix for new construction views
- Include level name in floor plans
- Use directional suffixes for elevations (NORTH & EAST, SOUTH & WEST)
