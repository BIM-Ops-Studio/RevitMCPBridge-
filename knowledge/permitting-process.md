# Permitting Process

## Permit Types

### Building Permits
| Type | Scope |
|------|-------|
| New construction | Complete new building |
| Addition | Increasing building area |
| Alteration | Modifying existing |
| Renovation | Updating existing |
| Demolition | Removing structure |
| Shell/Core | Base building only |
| Tenant improvement | Interior fit-out |

### Trade Permits
| Permit | Scope |
|--------|-------|
| Electrical | Wiring, panels, fixtures |
| Plumbing | Piping, fixtures, water heaters |
| Mechanical | HVAC, ductwork, equipment |
| Gas | Gas piping, appliances |
| Fire alarm | Detection, notification |
| Fire sprinkler | Suppression systems |
| Low voltage | Data, security, AV |

### Site Permits
| Permit | Scope |
|--------|-------|
| Grading | Earth moving |
| Paving | Roads, parking |
| Stormwater | Drainage systems |
| Utility connection | Water, sewer, electric |
| Right-of-way | Work in public ROW |
| Tree removal | Protected trees |
| Environmental | Wetlands, buffers |

### Other Permits
| Permit | Purpose |
|--------|---------|
| Sign permit | New or modified signage |
| Fence permit | Fences over height limit |
| Pool permit | Swimming pools, spas |
| Elevator | New or modified lifts |
| Generator | Standby power |
| Roofing | Re-roofing projects |
| Solar | PV installations |

## Submittal Requirements

### Architectural Drawings
| Sheet | Required Content |
|-------|------------------|
| Cover | Project info, index, location |
| Site plan | Building location, setbacks |
| Floor plans | All levels, dimensions |
| Elevations | All sides, heights |
| Sections | Building sections |
| Details | Wall, roof, connections |
| Schedules | Doors, windows, finishes |

### Structural Drawings
| Sheet | Required Content |
|-------|------------------|
| Foundation plan | Footings, slabs, piles |
| Framing plans | Each level |
| Roof framing | Trusses, rafters |
| Details | Connections, reinforcing |
| Schedules | Beams, columns |
| Calculations | Load path, design |

### MEP Drawings
| Discipline | Required Content |
|------------|------------------|
| Electrical | Panel schedules, circuits, lighting |
| Plumbing | Isometrics, fixture schedules |
| Mechanical | Duct layouts, equipment schedules |
| Fire protection | Sprinkler layouts, calculations |

### Supporting Documents
| Document | Purpose |
|----------|---------|
| Survey | Property boundaries, topography |
| Geotechnical report | Soil conditions |
| Energy calculations | Code compliance |
| Structural calculations | Design verification |
| Product data | Specific materials |
| Manufacturer approvals | FL Product Approval |

|------|----------|

### Review Disciplines
| Reviewer | Focus |
|----------|-------|

| Structural | Structural adequacy |
| Mechanical | HVAC, energy |
| Electrical | Electrical systems |
| Plumbing | Plumbing systems |
| Fire | Fire protection, life safety |

| Environmental | Stormwater, wetlands |

### Common Review Comments
| Category | Typical Issues |
|----------|----------------|
| Life safety | Egress, fire separation |
| Accessibility | ADA compliance |
| Structural | Load path, connections |
| Energy | Compliance documentation |
| Fire | Separation, sprinklers |

### Response to Comments
| Action | Description |
|--------|-------------|
| Revise drawings | Update per comments |
| Provide calculations | Support design |
| Clarify | Add notes, details |
| Request meeting | Complex issues |
| Appeal | If disagreement |

## Certificate of Occupancy

### CO Requirements
| Requirement | Description |
|-------------|-------------|
| All inspections passed | Building, trade, fire |
| Utility connections | Water, sewer, electric approved |
| Fire department approval | Alarm, sprinkler tested |
| Health department | Food service, pools |
| Planning final | Landscaping, parking |
| Fees paid | All outstanding fees |

### CO Types
| Type | Use |
|------|-----|
| Certificate of Occupancy | Permanent, final |
| Temporary CO (TCO) | Limited duration, conditions |
| Certificate of Completion | Non-occupancy (e.g., parking) |
| Change of Use CO | New use of existing building |

### TCO Conditions
| Condition | Description |
|-----------|-------------|
| Outstanding items | Punch list, minor work |
| Seasonal items | Landscaping, paving |
| Phased occupancy | Portion of building |
| Time limit | 30-90 days typical |
| Bond/deposit | May be required |

## Timelines

### Typical Permit Timeline
| Phase | Duration |
|-------|----------|
| Application prep | 1-2 weeks |
| Submittal | 1 day |
| Initial review | 2-4 weeks |
| Revisions | 1-2 weeks |
| Re-review | 1-2 weeks |
| Permit issuance | 1-3 days |
| **Total** | **6-12 weeks** |

### Factors Affecting Timeline
| Factor | Impact |
|--------|--------|
| Project complexity | More review time |
| Jurisdiction workload | Delays possible |
| Completeness | Incomplete = longer |
| Variances needed | Additional hearings |
| Outside agencies | Army Corps, state |

### Expediting Strategies
| Strategy | Description |
|----------|-------------|
| Pre-application meeting | Identify issues early |
| Complete submission | All documents included |
| Experienced team | Fewer errors |
| Expedited review | Pay premium fee |
| Phased permits | Start early phases |
| Concurrent review | Trade permits parallel |

## Fees

### Fee Types
| Fee | Calculation |
|-----|-------------|
| Building permit | Per SF or valuation |
| Trade permits | Flat or per fixture |
| Plan review | 50-100% of permit fee |
| Re-review | Per occurrence |
| Expedited review | 50-100% premium |
| Impact fees | Per unit/SF |

### Typical Fee Ranges
| Project Type | Permit Fee Range |

| Commercial (small) | $5,000-20,000 |

| Tenant improvement | $1,000-10,000 |

|----------|---------------------|

## Common Issues

### Zoning Issues
| Issue | Resolution |
|-------|------------|
| Setback violation | Variance or redesign |
| Height exceeded | Variance or redesign |
| Parking shortage | Variance or shared parking |
| FAR exceeded | Reduce building area |
| Non-conforming | Special approval |

### Code Issues
| Issue | Resolution |
|-------|------------|
| Egress deficiency | Add exits, widen |
| Fire separation | Add rated assembly |
| Accessibility | Revise for compliance |
| Structural inadequate | Engineer redesign |
| Energy non-compliance | Upgrade envelope/systems |

### Process Issues
| Issue | Prevention |
|-------|------------|
| Incomplete submittal | Use checklist |
| Missed comments | Track all responses |
| Expired permit | Monitor validity period |
| Work without permit | Apply for after-the-fact |
| Changes during construction | Revision permit |

## After Permit Issuance

### Permit Validity
| Jurisdiction | Typical Validity |
|--------------|------------------|
| Florida (general) | 180 days to start |
| Extensions | 180 days each |
| Maximum extensions | 2-3 typical |
| Expired permit | Re-apply required |

### Maintaining Active Permit
| Requirement | Description |
|-------------|-------------|
| Continuous work | No long breaks |
| Inspections | Within validity period |
| Extension request | Before expiration |
| Fees | Pay extension fees |

### Changes During Construction
| Change Type | Action |
|-------------|--------|
| Minor | Note on permit set |
| Moderate | Revision permit |
| Major | Amendment review |
| Use change | New permit/CO |
