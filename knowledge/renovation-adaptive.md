### Alteration Levels
| Level | Scope | Requirements |
|-------|-------|--------------|
| Level 1 | < 50% of room | Maintain existing |
| Level 2 | > 50% of room | Upgrade work area |
| Level 3 | > 50% of building | Major upgrades |

## Structural Considerations

| Check | Purpose |
|-------|---------|
| Dead load | Existing weight |
| Live load | Current capacity |
| Lateral | Wind/seismic |

### Common Structural Issues
| Issue | Solution |
|-------|----------|
| Inadequate load capacity | Reinforce, redistribute |
| Deterioration | Repair, replace |
| Code deficiencies | Upgrade as required |
| Connection failure | Repair, replace |
| Settlement | Monitor, underpin |

### Adding Loads
| Strategy | Method |
|----------|--------|
| Lightweight materials | Reduce added weight |
| Load redistribution | New framing |
| Strengthening | Add members |
| Foundation upgrade | Underpin, add footings |

### Seismic Upgrades

|--------|-------------|
