# Modular and Prefabricated Construction

## Prefabrication Types

### Hierarchy of Prefabrication
| Level | Description | Example |
|-------|-------------|---------|
| Component | Individual parts | Trusses, precast panels |
| Panel | 2D assemblies | Wall panels, floor cassettes |
| Volumetric | 3D modules | Bathroom pods, rooms |
| Whole building | Complete structure | Modular buildings |

### Construction Methods
| Method | Off-site Work | On-site Work |
|--------|---------------|--------------|
| Stick-built | 0% | 100% |
| Panelized | 20-40% | 60-80% |
| Modular | 60-90% | 10-40% |
| Manufactured | 95%+ | 5% |
