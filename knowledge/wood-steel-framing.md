| Component | Description |
|-----------|-------------|
| Sole plate | Bottom plate, on subfloor |

| Double top plate | Required for bearing walls |
| King stud | Full stud at opening |
| Jack stud (trimmer) | Supports header |
| Cripple stud | Short stud above/below opening |

## Fire Protection

### Wood Fire-Blocking
| Location | Requirement |
|----------|-------------|
| Vertical | 10' max height |
| Horizontal | At floor/ceiling |
| Concealed spaces | Subdivide |
| Stair stringers | Block at top |
| Soffits | At exterior walls |

### Steel Fireproofing
| Method | Rating Available |
|--------|------------------|
| Spray-applied | 1-4 hours |
| Intumescent paint | 1-2 hours |
| Board systems | 1-4 hours |
| Concrete encasement | 2-4 hours |

### Type V Construction (Wood)
| Element | Requirement |
|---------|-------------|
| Exterior walls | 1-hour or less |
| Interior bearing | 1-hour or less |
| Floor | 1-hour or less |
| Roof | No requirement |
| Height limit | 60' |

## Hurricane/High Wind

### Florida Requirements
| Component | Requirement |
|-----------|-------------|
| Roof sheathing | 8d @ 6" edge, 6" field |
| Wall sheathing | 8d @ 4" edge, 6" field |
| Roof-to-wall | Hurricane clips every truss |
| Wall-to-floor | Straps at each level |
| Sill-to-foundation | Anchor bolts @ 4' O.C. |

### High-Wind Connectors
| Location | Connector |
|----------|-----------|
| Roof to wall | H2.5 or H10 |
| Wall to wall | MST or similar |
| Wall to foundation | MASA or HD |
| Continuous tie | Threaded rod |
