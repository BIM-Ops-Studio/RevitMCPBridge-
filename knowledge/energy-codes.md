# Energy Codes and Compliance

## Code Framework

### Primary Energy Codes
| Code | Jurisdiction | Application |
|------|--------------|-------------|
| IECC | Model code (adopted by states) | Commercial and residential |
| ASHRAE 90.1 | Referenced standard | Commercial buildings |
| Title 24 | California | All buildings in CA |
| NYCECC | New York City | NYC buildings |
| Stretch codes | Various localities | Beyond base code |

### IECC Editions
| Edition | Key Changes |
|---------|-------------|
| 2021 IECC | ERI path, solar-ready, EV-ready |
| 2018 IECC | Increased envelope, lighting efficiency |
| 2015 IECC | Air barrier requirements |
| 2012 IECC | Blower door testing required |

### Adoption by State
| Adoption Level | States |
|----------------|--------|
| 2021 IECC | CA, WA, OR, VT, MA |
| 2018 IECC | NY, NJ, MD, VA, CO |
| 2015 IECC | TX, FL, GA, NC |
| 2012 or older | Many southern/rural states |

## Compliance Paths

### IECC Commercial Compliance
| Path | Description | Best For |
|------|-------------|----------|
| Prescriptive | Meet each requirement individually | Simple buildings |
| Trade-off (COMcheck) | Trade between components | Moderate flexibility |
| Energy Cost Budget | Whole-building simulation | Maximum flexibility |

### IECC Residential Compliance
| Path | Description | Best For |
|------|-------------|----------|
| Prescriptive | Meet all requirements | Standard construction |
| UA Trade-off | Trade envelope components | Design flexibility |
| Simulated Performance | Energy modeling | Custom designs |
| ERI (HERS) | Energy Rating Index | High-performance |

### ASHRAE 90.1 Compliance
| Path | Description |
|------|-------------|
| Prescriptive | Section-by-section requirements |
| Trade-off (ECB) | Energy Cost Budget method |
| Performance (PRM) | Performance Rating Method |
| Appendix G | Detailed simulation rules |

## Air Barrier Requirements

### Commercial Air Barrier
| Requirement | Standard |
|-------------|----------|
| Continuous air barrier | Required all climate zones |
| Material air permeance | ≤ 0.004 cfm/sf at 0.3" w.c. |
| Assembly air leakage | ≤ 0.04 cfm/sf at 0.3" w.c. |
| Building air leakage | ≤ 0.40 cfm/sf at 0.3" w.c. |

### Residential Air Sealing
| Climate Zone | Maximum ACH50 |
|--------------|---------------|
| 1-2 | 5 ACH50 |
| 3-8 | 3 ACH50 |

### Critical Air Barrier Locations
- Window and door frames
- Utility penetrations
- Rim joists
- Top plates
- Recessed lighting
- HVAC registers
- Electrical boxes

| Retail | 1.05 |
| School | 0.81 |

| Equipment | Efficiency | Standard |
|-----------|------------|----------|
| Split AC (< 65 kBtu) | 14 SEER | Federal minimum |

## Common Compliance Issues

### Design Phase
| Issue | Solution |
|-------|----------|
| Window area too high | Reduce glazing or improve U/SHGC |
| Insufficient insulation | Add continuous insulation |
| Missing air barrier | Specify continuous barrier |
| Lighting over budget | Reduce LPD, add controls |

### Construction Phase
| Issue | Solution |
|-------|----------|
| Thermal bridging | Specify thermal breaks |
| Air barrier gaps | Continuous inspection |
| Duct leakage | Pre-cover testing |
| Missing insulation | Photo documentation |

### Inspection Points
| Trade | Inspection |
|-------|------------|
| Envelope | Insulation before cover |
| Air barrier | Continuity check |
| Mechanical | Equipment verification |
| Electrical | Lighting controls |
| Final | Blower door (if req'd) |
