# Mixed-Use Development

## Development Types

### Vertical Mixed-Use
| Configuration | Description |
|---------------|-------------|
| Retail over residential | Ground floor retail, units above |
| Office over retail | Ground floor retail, offices above |
| Hotel over retail | Ground floor retail, hotel above |
| Residential over office | Offices lower, residential upper |
| Stacked uses | Multiple distinct zones |

### Horizontal Mixed-Use
| Configuration | Description |
|---------------|-------------|
| Urban village | Multiple buildings, mixed uses |
| Lifestyle center | Retail + office + residential |
| Town center | Civic + commercial + residential |
| Transit-oriented | Centered on transit station |

### Use Combinations
| Primary | Secondary | Tertiary |
|---------|-----------|----------|
| Residential | Retail | Office |
| Office | Retail | Restaurant |
| Hotel | Restaurant | Retail |
| Residential | Parking | Retail |

## Planning Considerations

### Site Organization
| Element | Consideration |
|---------|---------------|
| Street frontage | Retail activation |
| Building entrances | Separate by use |
| Parking access | Minimize conflicts |
| Loading | Screened, separate |
| Open space | Public vs private |
| Pedestrian flow | Through-block connections |

| Location | Typical FAR |
|----------|-------------|

| Downtown | 8.0-20.0 |

| Use | Ratio | Peak Time |
|-----|-------|-----------|
| Residential | 1.0-1.5/unit | Night |
| Office | 3.0/1,000 SF | Day |
| Retail | 4.0/1,000 SF | Evening/weekend |
| Restaurant | 10.0/1,000 SF | Evening |

### Shared Parking Reduction
| Time | Dominant Use |
|------|--------------|
| Weekday 8am-6pm | Office |
| Weekday 6pm-10pm | Retail/Restaurant |
| Weekend | Retail/Restaurant |
| Night | Residential/Hotel |

|-------|----------|

| Electrical | Separate services/meters |

| Fire alarm | Single system, zoned |

### Egress Separation
| Requirement | Description |

|-----------------|--------------------------|

| R-2 / M (Retail) | 1-hour |
| R-2 / A-2 (Restaurant) | 1-hour |
| R-2 / S-2 (Parking) | 1-hour |

|--------|--------------|

| Sprinklers | Throughout |
| Standpipes | Class I |
| Fire command | Ground floor |

| Issue | Solution |
|-------|----------|
