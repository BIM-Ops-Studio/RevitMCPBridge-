| Type | Use | Pros | Cons |
|------|-----|------|------|
| LCD/LED | General | Bright, affordable | Limited size |
| OLED | Premium | Contrast, thin | Cost |

| LED wall | High-end | Seamless, bright | Very expensive |

|------------|-------|
| Ultra-short | < 0.4:1 |
| Short throw | 0.4-1.0:1 |
| Standard | 1.5-2.5:1 |

### Lumens by Application
| Application | Lumens |
|-------------|--------|
| Small conference | 3,000-4,000 |
| Medium conference | 4,000-6,000 |
| Training room | 5,000-7,000 |
| Auditorium | 8,000-15,000 |

### Screen Types
| Type | Use |
|------|-----|
| Matte white | General purpose |
| High contrast | Ambient light |
| Acoustic | Behind speakers |
| Rear projection | Special applications |
| Motorized | Multi-use spaces |

### Screen Sizing
```
Image Width = Throw Distance / Throw Ratio

Example:
15' throw / 1.5 ratio = 10' wide image
```
