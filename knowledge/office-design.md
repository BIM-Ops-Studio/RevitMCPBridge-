# Office Building Design

## Space Planning Standards

### Space Allocation
| Position | Private Office | Workstation |
|----------|---------------|-------------|
| Executive | 200-400 SF | - |
| Senior manager | 150-200 SF | - |
| Manager | 120-150 SF | - |
| Professional | 80-120 SF | 48-64 SF |
| Administrative | - | 36-48 SF |
| Technical | - | 48-64 SF |

### Common Areas
| Space | SF per Person | Notes |
|-------|---------------|-------|
| Conference (large) | 25-30 SF | 20+ people |
| Conference (small) | 20-25 SF | 6-12 people |
| Huddle room | 15-20 SF | 2-4 people |
| Break room | 15-20 SF | Full service |
| Coffee bar | 5-10 SF | Basic service |
| Reception | 50-100 SF | Plus waiting |

### Support Spaces
| Space | Allocation |
|-------|------------|
| Copy/print | 1 per 20-40 people |

## Tenant Improvement

### Shell vs. Tenant Work
| Shell (Landlord) | Tenant Improvement |
|------------------|-------------------|
| Base building structure | Partitions |
| Core restrooms | Private restrooms |
| Base HVAC to floor | Ductwork/diffusers |
| Main electrical | Circuits to devices |
| Fire protection main | Adjust heads |
| Elevator lobbies | Reception, finishes |

### Standard Tenant Work Letter
- Partitions (LF per SF)
- Doors (number included)
- Electrical outlets per SF
- Data drops per SF
- HVAC zones
- Light fixtures per SF
- Flooring allowance

### Typical TI Allowance
| Market | New Construction | 2nd Gen |
|--------|-----------------|---------|
| Class A | $50-80/SF | $30-50/SF |
| Class B | $30-50/SF | $20-35/SF |
| Class C | $15-30/SF | $10-20/SF |
