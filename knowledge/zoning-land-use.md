## Development Standards

### Setbacks
| Type | Description |
|------|-------------|
| Front | Distance from front property line |
| Side | Distance from side property line |
| Rear | Distance from rear property line |
| Corner | On corner lots, both frontages |
| Street side | Side yard abutting street |

### Typical Residential Setbacks
| Zone | Front | Side | Rear |

| Multi-family | 25'-35' | 15'-25' | 25' |

## Approval Processes

### By-Right Development
| Characteristic | Process |
|----------------|---------|
| Meets all standards | Staff approval |
| No variances | Building permit |
| Typical timeline | 2-4 weeks |

### Site Plan Review
| Trigger | Requirements |
|---------|--------------|
| Commercial development | Site plan |
| Multi-family | Site plan |
| PUD | Master plan |
| Size threshold | Traffic study |

### Conditional Use/Special Exception
| Purpose | Process |
|---------|---------|
| Uses permitted with conditions | Public hearing |
| May impose conditions | Board approval |
| Typical timeline | 2-4 months |

### Variance
| Type | Standard |
|------|----------|
| Dimensional | Setback, height, coverage |
| Use | Non-conforming use |
| Hardship required | Unique circumstance |
| Not self-created | Cannot be applicant's doing |

### Rezoning
| Process | Timeline |
|---------|----------|
| Application | Submit request |
| Staff review | 2-4 weeks |
| Planning commission | Public hearing |
| City/County commission | Final hearing |
| Total | 3-6 months |

## Signage

| Projecting | Perpendicular to wall |

## Non-Conforming Uses

### Legal Non-Conforming
| Status | Description |
|--------|-------------|
| Grandfathered | Existed before zoning change |
| May continue | Current use permitted |
| Cannot expand | Except as allowed |
| May lose status | If abandoned |

### Expansion Rules
| Situation | Typical Rule |
|-----------|--------------|
| Minor expansion | 10-25% may be allowed |
| Major expansion | Usually prohibited |
| Damage/destruction | 50% rule common |
| Abandonment | 6-12 months typical |

### Amortization
| Concept | Description |
|---------|-------------|
| Purpose | Phase out non-conforming |
| Timeline | Varies by use |
| Compensation | May be required |
| Legal challenges | Common |
