# Resilient Design

## Resilience Overview

### Definition
Resilient design creates buildings and communities that can withstand, adapt to, and rapidly recover from disruptions including natural disasters, climate change impacts, and infrastructure failures.

### Resilience Framework
| Phase | Goal |
|-------|------|
| Prepare | Reduce vulnerability |
| Withstand | Survive the event |
| Recover | Return to function |
| Adapt | Improve for future |

### Hazard Categories
| Category | Examples |
|----------|----------|
| Climate | Heat, drought, precipitation |
| Weather | Hurricane, tornado, flood |
| Geologic | Earthquake, landslide |
| Fire | Wildfire, urban fire |
| Infrastructure | Power outage, water loss |
| Human | Pandemic, civil unrest |

| Strategy | Implementation |
|----------|----------------|

## Flood Resilience

### Flood Zone Classifications
| Zone | Description | Requirements |
|------|-------------|--------------|
| VE | Coastal high hazard | Elevated, breakaway walls |
| AE | Base flood w/ BFE | Elevated or floodproofed |
| AO | Sheet flow | Elevated above depth |
| X (shaded) | 500-year | Consider protection |
| X (unshaded) | Minimal flood | Standard construction |

### Elevation Strategies
| Approach | Application |
|----------|-------------|
| Fill | Raise entire site |
| Foundation walls | Crawlspace above BFE |
| Piers/piles | Coastal, deep flood |
| Stilts | V-zone required |
| Living shoreline | Coastal buffer |

### Freeboard
| Jurisdiction | Above BFE |
|--------------|-----------|
| NFIP minimum | 0" |
| Recommended | +1' to +2' |
| ASCE 24 (critical) | +2' to +3' |
| Sea level rise | +1' to +3' additional |

### Flood-Resistant Materials
| Zone | Requirement |
|------|-------------|
| Below BFE | Class 4-5 flood damage resistant |
| Utilities | Elevated or submersible |
| Insulation | Closed-cell only |
| Flooring | Concrete, tile, vinyl |
| Walls | CMU, pressure-treated, fiber cement |

### Floodproofing Types
| Type | Description | Limitations |
|------|-------------|-------------|
| Dry | Watertight barriers | ≤ 3' depth |
| Wet | Allow flooding, resistant materials | Non-residential |
| Temporary | Deployable barriers | Requires warning time |

## Fire Resilience (Wildfire)

### Wildfire Zones
| Zone | Distance | Treatment |
|------|----------|-----------|
| Zone 0 | 0-5' | Non-combustible |
| Zone 1 | 5-30' | Fire-resistant, lean |
| Zone 2 | 30-100' | Reduced fuels |
| Zone 3 | 100-200' | Managed vegetation |

### Ignition-Resistant Construction
| Component | Strategy |
|-----------|----------|
| Roofing | Class A (metal, tile, composite) |
| Walls | Non-combustible, 6" above grade |
| Vents | 1/8" mesh, ember-resistant |
| Eaves | Enclosed, non-combustible |
| Decks | Non-combustible or ignition-resistant |
| Windows | Multi-pane, tempered |

### Defensible Space
| Element | Treatment |
|---------|-----------|
| Vegetation | Spacing, low-flammability species |
| Mulch | Non-combustible within 5' |
| Fencing | Non-combustible within 5' |
| Storage | No combustibles against building |
| Trees | Limbed up, spaced crowns |
