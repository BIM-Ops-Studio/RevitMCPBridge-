# Millwork Standards

## Casework Types

### AWI Quality Grades
| Grade | Description | Use |
|-------|-------------|-----|
| Economy | Minimum acceptable | Back-of-house, utility |
| Custom | Standard commercial | Office, retail |
| Premium | Highest quality | Executive, high-end |

### Construction Types
| Type | Description | Use |
|------|-------------|-----|
| Frameless (European) | No face frame, full overlay doors | Modern, efficient |
| Face Frame | Traditional frame construction | Traditional, residential |
| Flush | Flat front, concealed hinges | Modern, commercial |

## Finish Types

### Factory Finishes
| Finish | Description | Durability |
|--------|-------------|------------|
| Catalyzed lacquer | Clear or pigmented | Excellent |
| Catalyzed varnish | Clear, high build | Excellent |
| UV-cured | Factory applied, very durable | Excellent |
| Pre-catalyzed lacquer | Simpler finish | Good |

### Paint Finishes
| Sheen | Use | Cleaning |
|-------|-----|----------|
| Flat | Never for millwork | N/A |
| Satin | Residential, low use | Good |
| Semi-gloss | Commercial, moderate use | Very good |
| High gloss | High-end, modern | Excellent |

### Stain Guidelines
- Sample on actual material
- Stain + clear coat (3 coats min)
- Consider pre-stain conditioner for softwoods
- Match across all elements (doors, frames, panels)

## Shop Drawing Requirements

### Information to Include
- Plan view with dimensions
- Elevations of each cabinet run
- Sections through cabinets
- Hardware schedule
- Material/finish schedule
- Edge banding specifications
- Counter material and edge
- Toe kick details
- Scribe allowances

### Dimensions to Verify
- Field verify all dimensions before fabrication
- Allow 1/8" scribe at walls
- Note filler locations
- Indicate outlet/switch cutouts
- Show appliance locations and sizes
- Indicate plumbing locations

## Installation Standards

### Substrate Requirements
| Cabinet Location | Requirement |
|------------------|-------------|
| Wall hung | Blocking or heavy-duty anchors |
| Base on slab | Shim level, silicone at floor |
| Island | Anchor to floor, level |

### Reveal/Alignment Tolerances
| Feature | Tolerance |
|---------|-----------|
| Drawer fronts | 1/16" reveal |
| Door alignment | 1/16" reveal |
| Level | 1/8" in 8' |
| Plumb | 1/8" in 8' |
| Adjacent faces | Flush within 1/32" |

### Scribe Allowances
- 1/4" to 1/2" at walls
- Scribe strip on exposed ends
- Filler strips at tight conditions
- Cover plate at electrical

## Specialty Millwork

### Reception Desks
- Transaction top: 42"-44" (standing)
- Work surface: 29"-30" (seated)
- ADA section: 34" max
- Knee space: 30"W × 27"H × 19"D

### Built-in Bookcases
| Shelf Span | Material/Thickness |
|------------|-------------------|
| Up to 24" | 3/4" minimum |
| 24"-36" | 3/4" with support or 1" |
| Over 36" | Add vertical divider |

### Window Seats
- Height: 18"-20" (seating)
- Depth: 18"-24"
- Hinged top or drawer access
- Support for 300 lbs

### Closet Systems
| Element | Dimension |
|---------|-----------|
| Hanging rod | 66"-72" for long, 42" for double |
| Shelf above rod | 12" deep, 2" above rod |
| Shoe shelf | 12" deep, 8"-12" between |
| Drawer stack | 24"-36" wide |

## Cost Factors

### Cost Drivers (High Impact)
- Exotic wood species
- High-gloss or lacquer finish
- Curved or radius work
- Glass doors
- Pull-out accessories
- Soft-close hardware
- Premium hardware (brand name)

### Cost Savers
- Standard sizes
- Frameless construction
- Melamine interiors
- Paint-grade materials
- Standard overlay doors
- Economy hardware
