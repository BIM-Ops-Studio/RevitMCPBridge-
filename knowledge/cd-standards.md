# Construction Document Standards

## Sheet Numbering System

### Standard Format: [Discipline][Type][Sequence]

| Prefix | Discipline |
|--------|------------|
| G | General (cover, code, life safety) |
| C | Civil / Site |
| L | Landscape |
| A | Architectural |
| S | Structural |
| M | Mechanical |
| P | Plumbing |
| E | Electrical |
| F | Fire Protection |

### Architectural Sheet Types
| Number | Content |
|--------|---------|
| A0.X | General - Symbols, Abbreviations, Notes |
| A1.X | Floor Plans |
| A2.X | Elevations |
| A3.X | Sections |
| A4.X | Large Scale Plans (stairs, bathrooms, kitchens) |
| A5.X | Schedules (doors, windows, finishes) |
| A6.X | Interior Elevations |
| A7.X | Reflected Ceiling Plans |
| A8.X | Roof Plans |
| A9.X | Details |

## Drawing Scales

| Drawing Type | Residential Scale | Commercial Scale |
|--------------|-------------------|------------------|
| Site Plan | 1" = 20'-0" | 1" = 30'-0" or 1" = 40'-0" |
| Floor Plans | 1/4" = 1'-0" | 1/8" = 1'-0" |
| Elevations | 1/4" = 1'-0" | 1/8" = 1'-0" |
| Building Sections | 1/4" = 1'-0" | 1/8" = 1'-0" |
| Wall Sections | 3/4" = 1'-0" or 1" = 1'-0" | 1/2" = 1'-0" or 3/4" = 1'-0" |
| Details | 1-1/2" = 1'-0" or 3" = 1'-0" | 1" = 1'-0" or 1-1/2" = 1'-0" |
| Enlarged Plans | 1/2" = 1'-0" | 1/4" = 1'-0" |

## Title Block Requirements

### Required Information
- Project Name
- Project Address
- Project Number
- Client Name
- Architect Name and Address
- Architect License Number
- Sheet Number
- Sheet Title
- Date
- Scale (or "AS NOTED")
- Drawn By initials
- Checked By initials
- Revision block (date, description, delta number)

### Sheet Sizes
| Size | Dimensions | Typical Use |
|------|------------|-------------|
| ARCH D | 24" x 36" | Standard full-size |
| ARCH E | 36" x 48" | Large commercial |
| ARCH C | 18" x 24" | Half-size prints |
| ANSI B | 11" x 17" | Submittals |

## View Naming Conventions

### Floor Plans
- "FIRST FLOOR PLAN" (not "Level 1" or "Floor 1")
- "SECOND FLOOR PLAN"
- "GROUND FLOOR PLAN" (if below grade entry)
- "BASEMENT PLAN" or "LOWER LEVEL PLAN"

### Ceiling Plans
- "FIRST FLOOR REFLECTED CEILING PLAN" or "FIRST FLOOR RCP"

### Elevations
- "NORTH ELEVATION", "SOUTH ELEVATION", etc.
- Or by feature: "FRONT ELEVATION", "REAR ELEVATION"

### Sections
- "BUILDING SECTION 1", "BUILDING SECTION 2"
- "WALL SECTION AT EXTERIOR"
- "WALL SECTION AT INTERIOR"

## Minimum Sheet Set by Phase

### Schematic Design (SD)
- Site plan
- All floor plans
- 4 building elevations
- 1-2 building sections
- 3D views/renderings
- Area calculations

### Design Development (DD)
All SD items plus:
- Enlarged plans (stairs, bathrooms, kitchens)
- Interior elevations (key rooms)
- Wall sections (typical conditions)
- Door schedule (preliminary)
- Window schedule (preliminary)
- Room finish schedule
- Outline specifications

### Construction Documents (CD)
All DD items plus:
- Complete dimensions on all plans
- All details (roof, wall, foundation, misc)
- Complete schedules
- General notes
- Code analysis
- Life safety plans
- Accessibility compliance drawings
