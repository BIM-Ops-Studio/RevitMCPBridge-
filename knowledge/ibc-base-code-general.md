|-------|-------------|----------|

| A-4 | Indoor viewing of sports | Arenas, skating rinks |

- Offices, banks, educational above 12th grade

| Group | Description |
|-------|-------------|

|-------|-------------|

| Group | Description |
|-------|-------------|

| Group | Description |
|-------|-------------|

| Group | Description |
|-------|-------------|

### Area Increases

**Frontage Increase (507.2)**:
```
If = 100 × (F/P - 0.25) × W/30

Where:
F = Building perimeter fronting on open space
P = Total building perimeter
W = Width of open space (20' min, 30' max)
```

**Sprinkler Increase (506.3)**:
- Multi-story: 200% increase (3× base area)
- Single-story: 300% increase (4× base area)
- Unlimited area: Available for certain combinations

## Fire Barriers, Walls, and Partitions (Chapter 7)

### Fire Walls (706)
- **Rating**: 2-4 hours (per Table 706.4)
- **Purpose**: Divide building into separate structures
- **Structural stability**: Must remain standing if structure on one side collapses

### Fire Barriers (707)
- **Purpose**: Separate occupancies, exit enclosures, shafts
- **Ratings**: Per specific use

| Use | Rating |
|-----|--------|
| Shaft enclosure (4+ stories) | 2 hour |
| Shaft enclosure (< 4 stories) | 1 hour |
| Exit enclosure (4+ stories) | 2 hour |
| Exit enclosure (< 4 stories) | 1 hour |
| Exit passageway | Same as exit enclosure |
| Horizontal exit | 2 hour |
| Incidental use separation | Per Table 509 |
| Occupancy separation | Per Table 508.4 |

### Fire Partitions (708)
- **Rating**: 1 hour (or less per specific use)
- **Use**: Dwelling/sleeping unit separation, corridor walls

### Smoke Barriers (709)
- **Rating**: 1 hour
- **Continuous**: Floor to underside of floor/roof above

### Smoke Partitions (710)
- **Rating**: Not required (smoke-tight only)
- **Use**: I-2 corridor walls, elevator lobbies

## Mixed Occupancy (508)

### Separated Uses
- Each occupancy must meet its own requirements
- Fire barrier between occupancies per Table 508.4

### Non-Separated Uses
- Building treated as most restrictive occupancy for:
  - Height
  - Area
  - Construction type
  - Fire protection

### Accessory Occupancies
- Max 10% of floor area
- Aggregate max 10% of building
- No separation required (except H, I-1, I-2)

## Special Detailed Requirements (Chapter 4)

### High-Rise Buildings (403)
**Applies when**: Building floor > 75' above fire department access

**Required**:
- Automatic sprinklers throughout
- Fire alarm and detection
- Emergency voice/alarm communication
- Fire command center
- Central control station
- Smokeproof enclosures OR pressurized exit stairs
- Fire service access elevators
- Standby power and emergency power
- Luminous egress path markings (75'-420' above grade)

### Atriums (404)
**Required**:
- Automatic sprinklers
- Smoke control system
- Fire barriers at atrium openings (or alternative)
- Standby power for smoke control

### Underground Buildings (405)
**Applies when**: Lowest occupied floor > 30' below exit discharge

**Required**:
- Automatic sprinklers
- Standby power
- Smoke control
- Compartmentalization

### Covered/Open Mall Buildings (402)
- Anchor building separation
- Tenant separation
- Exit requirements
- Automatic sprinklers

## General Building Limitations Summary

### Quick Reference - What Can I Build?

**Unlimited Height/Area** (Type I-A):
- Any occupancy
- No sprinkler increase needed
- High-rise requirements apply above 75'

**Common Wood Construction** (Type V-B):
- Max 40' height
- 1-2 stories (varies by occupancy)
- Base area ~9,000 SF (can triple with sprinklers)

**Typical Multi-Family** (Type V-A):
- 3 stories, 50' height
- 12,000 SF per floor
- Can reach 4 stories/36,000 SF with sprinklers
