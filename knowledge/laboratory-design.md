## Biosafety Levels

### BSL-1
| Feature | Requirement |
|---------|-------------|
| Containment | Open bench |
| HVAC | General building |
| Access | Unrestricted |
| PPE | Lab coat, gloves |
| Waste | Standard |

### BSL-2
| Feature | Requirement |
|---------|-------------|
| Containment | BSC for aerosols |
| HVAC | Directional airflow |
| Access | Controlled |
| PPE | Lab coat, gloves, face |
| Waste | Decontaminated |

### BSL-3
| Feature | Requirement |
|---------|-------------|
| Containment | All work in BSC |
| HVAC | 100% exhaust, HEPA |
| Access | Restricted, airlock |
| PPE | Respirator possible |
| Waste | Autoclaved on-site |

### BSL-4
| Feature | Requirement |
|---------|-------------|
| Containment | Suit or Class III BSC |
| HVAC | Double HEPA, incineration |
| Access | Shower out, isolated |
| PPE | Positive pressure suit |
| Waste | Chemical + autoclave |
