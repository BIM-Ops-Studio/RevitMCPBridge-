## Stair Standards

### Stair Dimensions
| Element | Code Minimum | Comfortable |
|---------|--------------|-------------|
| Width | 36" | 42" |
| Riser height | 7-3/4" max | 7" |
| Tread depth | 10" min | 11" |
| Headroom | 6'-8" | 7'-0" |
| Landing depth | Width of stair | Width + 6" |

### Stair Run Calculation
For floor-to-floor of 9'-0" (108"):
- At 7" risers: 108 ÷ 7 = 15.4 → Use 16 risers at 6.75"
- Treads = Risers - 1 = 15 treads
- Run = 15 × 11" = 165" = 13'-9"

### Handrail Heights
| Location | Height |
|----------|--------|
| At stairs | 34" - 38" from nosing |
| At landings | 34" - 38" AFF |
| Guards (open side) | 36" residential / 42" commercial |

## Window Standards

### Window Sizing
| Room | Glazing Area | Sill Height | Head Height |
|------|--------------|-------------|-------------|
| Bedroom | 8% of floor area (egress) | 24"-36" | 6'-8" |
| Living room | 8% of floor area | 18"-30" | 6'-8" |
| Kitchen | 8% of floor area | 42" (above counter) | 6'-8" |
| Bathroom | Varies | 48"+ (privacy) | 6'-8" |

### Egress Window Requirements (Bedrooms)
- Minimum opening: 5.7 SF
- Minimum width: 20"
- Minimum height: 24"
- Maximum sill height: 44" AFF
