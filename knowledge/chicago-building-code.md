# Chicago Building Code - Key Amendments

**Based on**: Chicago Building Code (2022)
**Base Code**: IBC 2018 with Chicago amendments
**Authority**: Chicago Department of Buildings (DOB)
**Municipal Code**: Title 14 (Building Code), Title 15 (Fire Prevention)

---

## Quick Reference - Chicago vs. IBC

| Requirement | Chicago | IBC 2021 |
|-------------|---------|----------|
| High-rise threshold | 80' (not 75') | 75' |
| Sprinkler threshold (R-2) | 4+ stories or 80'+ | 3+ stories |
| Seismic design category | A or B | Varies |
| Snow load | 25 psf | 25 psf |
| Wind speed | 105-115 mph | 105-115 mph |
| Frost depth | 42" | 42" |
| Exit access travel (B) | 250' / 300' sprinklered | 200' / 300' sprinklered |
| Corridor fire rating | Per table | Per IBC table |

## High-Rise Buildings (Chicago Chapter 13-160)

### Definition
- **High-rise**: Occupied floor > **80 feet** above fire department access
- *Note*: This differs from IBC (75 feet)

### High-Rise Requirements

#### Fire Protection
| System | Requirement |
|--------|-------------|
| Sprinklers | Throughout building |
| Standpipes | Class I (2-1/2") |
| Fire alarm | Voice/alarm communication |
| Fire command center | Required |
| Smoke control | Required (or pressurized stairs) |

#### Egress
- **Smokeproof enclosures**: Required (or pressurized stairs)
- **Exit access**: From all floors to ground
- **Re-entry**: Every 5 floors maximum
- **Stairway communication**: Two-way to fire command

#### Structural
- **Fire resistance**: Per Class I construction
- **Secondary structural members**: 2-hour minimum
- **Protected openings**: All exterior within 30'

### Fire Safety Director
- **Required**: All high-rise buildings
- **Duties**: Emergency procedures, drills, coordination
- **Training**: City-approved program

## Fire Protection

### Sprinkler Requirements

#### When Required (Chicago Specific)
- **High-rise**: All (> 80')
- **R-2 (apartments)**: 4+ stories or 80'+
- **Assembly > 300 occupants**: Required
- **Basements > 2,500 sf**: Required
- **Mercantile > 12,000 sf**: Required
- **Group E (schools)**: Required

### Standpipe Requirements

| Building | Class | Size |
|----------|-------|------|
| High-rise | I | 2-1/2" |
| 4+ stories | I | 2-1/2" |
| > 10,000 sf per floor | I | 2-1/2" |
| Assembly | III | 1-1/2" and 2-1/2" |

### Fire Alarm

#### When Required
- **Assembly > 300 occupants**: Fire alarm
- **Business > 100 occupants**: Fire alarm
- **High-rise**: Voice alarm/evacuation
- **R-2**: Smoke detectors + notification
- **Covered mall**: Fire alarm

#### High-Rise Fire Alarm
- **Voice communication**: All floors
- **Fire command center**: Central location
- **Firefighter communication**: Portable radio enhancement
- **Two-way communication**: Stairwells, elevators

## Structural Requirements

### Seismic Design
- **Chicago SDC**: Typically A or B
- **Design not required**: For SDC A (most of Chicago)
- **Site-specific**: May be required for critical facilities
- **Short period (Ss)**: 0.07g - 0.10g
- **1-second period (S1)**: 0.02g - 0.04g

### Wind Loads
- **Basic wind speed**: 105-115 mph (Exposure B typical)
- **High-rise**: Enhanced wind analysis required
- **Cladding**: Component and cladding pressures

### Snow Loads
- **Ground snow load**: 25 psf
- **Flat roof snow load**: 25 psf minimum
- **Drift loads**: Per ASCE 7

### Foundation Requirements
- **Frost depth**: 42" minimum
- **Presumptive bearing**: Per soil type
- **Caissons**: Common for high-rise
- **Chicago hardpan**: Unique soil conditions

## Accessibility

### Chicago Accessibility
- Follows Illinois Accessibility Code (IAC)
- Based on ADA 2010 Standards with state amendments

### Key Requirements
| Element | Requirement |
|---------|-------------|
| Accessible entrance | At least 1 |
| Accessible parking | Per ADA table |
| Signage | Room signs with Braille |
| Toilet rooms | See Illinois requirements |
| Dwelling units | Per Fair Housing Act |

### R-2 (Apartments)
- **Ground floor units (no elevator)**: All accessible
- **Elevator buildings**: All units accessible (Type B)
- **Type A units**: 5% of total

## Mechanical

### HVAC Requirements
- Chicago Mechanical Code applies
- Based on IMC with local amendments
- Enhanced requirements for high-rise

### Elevator Requirements

| Building Type | Requirement |
|---------------|-------------|
| R-2 > 3 stories | 1 elevator minimum |
| B > 4 stories | 1 elevator minimum |
| High-rise | Multiple, fire service access |
| High-rise R-2 | Stretcher-capable elevator |

### Emergency Power
- **High-rise**: Required for elevators, fire pumps, egress lighting
- **Hospitals**: Full emergency power
- **Assembly > 300**: Emergency lighting

## Plumbing

### Chicago Plumbing Code
- Based on IPC with Chicago amendments
- Administered by Chicago DOB

### Fixture Requirements
Similar to IBC/IPC with local modifications:
- **Residential**: 1 WC, 1 lav, 1 tub/shower per unit
- **Commercial**: Per occupancy load

### Backflow Prevention
- **Required**: All cross-connections
- **ASSE standards**: Apply
- **Testing**: Annual required

## Energy Code

### Chicago Energy Conservation Code
- Based on IECC 2018
- Chicago Energy Benchmarking Ordinance
- Large buildings: Annual energy use reporting

### Energy Benchmarking Ordinance
- **Applies to**: Buildings > 50,000 sf
- **Reporting**: Annual to City
- **Public disclosure**: Required
- **Enforcement**: Fines for non-compliance

## Unique Chicago Requirements

### Chicago Fire Prevention Code (Title 15)
- **High-rise fire drills**: Annual required
- **Fire watch**: When systems impaired
- **Hot work permits**: Required
- **Assembly permits**: For large gatherings

### Landmarks
- **Commission on Chicago Landmarks**: Approval required
- **Historic districts**: Special review
- **Facade changes**: May require permit

### Planned Development
- **Large projects**: City Council approval
- **Public benefits**: Often negotiated
- **Design review**: May be required

### Green Roofs
- **Incentives**: Available for green roofs
- **Stormwater**: Credit for green roof installation
- **Cool roofs**: Energy code compliance option

## Quick Checklists

### New Construction (Chicago)
- [ ] Zoning analysis (FAR, height, setbacks)
- [ ] Building classification determined
- [ ] High-rise? (> 80')
- [ ] Fire protection per class/use
- [ ] Sprinkler requirements verified
- [ ] Standpipe requirements
- [ ] Accessibility per Illinois code
- [ ] Energy code compliance
- [ ] Structural per Chicago amendments
- [ ] CFD approval for fire systems

### High-Rise Checklist
- [ ] Smokeproof or pressurized stairs
- [ ] Class I standpipes
- [ ] Automatic sprinklers throughout
- [ ] Fire command center
- [ ] Voice alarm communication
- [ ] Two-way communication (stairs/elevators)
- [ ] Elevator recall
- [ ] Fire service access elevator
- [ ] Emergency power (elevators, pumps, egress)
- [ ] Fire safety director designation
- [ ] Secondary power for smoke control

### Renovation Checklist
- [ ] Existing building review
- [ ] Landmarks/historic review (if applicable)
- [ ] Change of use triggers
- [ ] Accessibility upgrades required?
- [ ] Fire protection upgrades required?
- [ ] Energy benchmarking compliance
- [ ] Asbestos/lead survey
