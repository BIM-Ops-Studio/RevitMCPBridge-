# Building Commissioning

## Commissioning Overview

### Commissioning Levels
| Level | Scope | Description |
|-------|-------|-------------|
| Fundamental (ASHRAE 90.1) | HVAC controls, lighting controls | Code minimum |
| Enhanced (LEED) | Above + envelope, renewables | LEED requirement |
| Total Building | All systems | Full performance |
| Ongoing | Continuous monitoring | Continuous Cx |

### Project Phases
| Phase | Activities |
|-------|------------|
| Pre-Design | Owner's Project Requirements (OPR) |
| Design | Basis of Design (BOD), design reviews |
| Construction | Submittal review, installation verification |
| Acceptance | Functional testing, training |
| Post-Occupancy | Seasonal testing, warranty support |

### Key Documents
| Document | Purpose | Author |
|----------|---------|--------|
| OPR | Owner's requirements | Owner/CxA |
| BOD | Design response to OPR | Design team |
| Cx Plan | Cx activities and schedule | CxA |
| Cx Specifications | Contractor requirements | CxA |
| Issues Log | Track deficiencies | CxA |
| Final Report | Summary and documentation | CxA |

## Retro-Commissioning

### Investigation Phase
| Activity | Purpose |
|----------|---------|
| Utility analysis | Baseline energy use |
| Document review | Original design intent |
| Staff interviews | Operational issues |
| Site survey | Current conditions |
| Trending | BAS data analysis |

### Common Findings
| Issue | Impact |
|-------|--------|
| Schedules wrong | Energy waste |
| Setpoints drifted | Comfort, energy |
| Sensors failed | Control accuracy |
| Dampers stuck | Ventilation, energy |
| Sequences changed | Performance degradation |

### Implementation
| Priority | Action |
|----------|--------|
| High | No/low cost, high impact |
| Medium | Low cost, medium impact |
| Low | Capital, future budget |

### Payback Metrics
| Metric | Typical |
|--------|---------|
| Simple payback | 1-3 years |
| Energy savings | 5-15% |
| Comfort improvement | Significant |
| Equipment life | Extended |
