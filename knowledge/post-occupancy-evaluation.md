# Post-Occupancy Evaluation (POE)

## POE Overview

### Definition
Post-Occupancy Evaluation is a systematic process of evaluating buildings after they have been occupied, focusing on how well they meet user needs and perform as designed.

### POE Types
| Type | Timing | Depth | Duration |
|------|--------|-------|----------|
| Indicative | 3-6 months | Quick overview | 1-2 days |
| Investigative | 1-2 years | Detailed study | 1-4 weeks |
| Diagnostic | 2-5 years | Comprehensive | 2-6 months |

### Benefits
| Stakeholder | Benefit |
|-------------|---------|
| Owner | Improved operations, future planning |
| Occupants | Better environment, productivity |
| Designers | Learning, improvement |
| Industry | Knowledge advancement |

## Indoor Environmental Quality

### Thermal Comfort
| Parameter | Standard |
|-----------|----------|
| Temperature | 68-76°F (ASHRAE 55) |
| Humidity | 30-60% RH |
| Air velocity | < 40 fpm |
| Radiant asymmetry | < 9°F |

### Air Quality
| Parameter | Standard |
|-----------|----------|
| CO2 | < 1,000 ppm |
| PM2.5 | < 35 µg/m³ |
| TVOC | < 500 µg/m³ |
| Formaldehyde | < 27 ppb |
| Ventilation | Per ASHRAE 62.1 |

### Lighting Quality
| Parameter | Standard |
|-----------|----------|
| Task illuminance | 30-50 FC |
| Daylight factor | 2-5% |
| Glare | < UGR 19 |
| Color rendering | CRI > 80 |
| Contrast ratio | 3:1 |

### Acoustic Quality
| Parameter | Standard |
|-----------|----------|
| Background noise | NC 35-45 |
| Speech privacy | SII < 0.2 |
| Reverberation | RT60 0.5-1.0 sec |
| Sound masking | 40-48 dBA |
