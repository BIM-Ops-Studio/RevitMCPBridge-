## Code Requirements

### Occupancy Classification
| Space | Classification |
|-------|----------------|
| Classrooms | E (Educational) |
| Assembly (> 50) | A-3 |
| Gymnasium | A-3 or A-4 |
| Cafeteria | A-2 |
| Kitchen | A-2 |

### Egress
| Requirement | Value |
|-------------|-------|
| Occupant load (classrooms) | 20 SF/person |
| Occupant load (assembly) | 15 SF/person |
| Exit width | 0.2"/occupant |
| Travel distance | 250' (sprinklered) |
| Dead end | 50' max (sprinklered) |

### Fire Protection
| Requirement | Application |
|-------------|-------------|
| Sprinklers | Required most jurisdictions |
| Fire alarm | Addressable, voice evacuation |
| Smoke detection | Corridors, commons |
| Extinguishers | Per code spacing |
