| In-line retail | 1,000-5,000 SF | Strip center, mall |
| Junior anchor | 5,000-15,000 SF | Large specialty |
| Anchor | 15,000-50,000 SF | Department store |
| Big box | 50,000-200,000 SF | Category killer |
| Supermarket | 40,000-65,000 SF | Grocery |
| Hypermarket | 150,000-250,000 SF | Superstore |

## Space Planning

### Retail Layout Types
| Layout | Description | Use |
|--------|-------------|-----|
| Grid | Parallel aisles | Grocery, convenience |
| Loop/racetrack | Forced path | Department stores |
| Free-form | Organic flow | Boutique, specialty |
| Spine | Central aisle | Strip stores |
| Mixed | Combination | Large format |

### Circulation
| Element | Minimum | Preferred |
|---------|---------|-----------|
| Main aisle | 5'-0" | 8'-10' |
| Secondary aisle | 4'-0" | 5'-6' |
| Between fixtures | 3'-6" | 4'-5' |
| Checkout queue | 4'-0" | 6'-0" |
| ADA path | 36" | 44" |

### Space Allocation
| Zone | % of Total |
|------|------------|
| Sales floor | 60-70% |
| Back of house | 15-25% |
| Checkout/service | 5-10% |
| Entry/vestibule | 2-5% |
| Fitting rooms | 3-5% (apparel) |

## Storefront Design

### Entry Types
| Type | Use |
|------|-----|
| Recessed entry | Weather protection, display |
| Flush entry | Minimal footprint |
| Angled entry | Draw customers in |
| Open front | Mall inline |
| Vestibule | Climate control |

### Storefront Heights
| Location | Height |
|----------|--------|
| Strip center | 12'-16' |
| Mall inline | 12'-14' |
| Anchor | 20'-30' |
| Street retail | 14'-18' |

### Glazing
| Element | Requirement |
|---------|-------------|
| Display windows | 6'-0" to 8'-0" clear |
| Vision glass | 80%+ of storefront |
| Spandrel | Minimal or none |
| Door | Full glass typical |
| Sill height | 0"-18" for display |

## Accessibility

### Entry Requirements
| Feature | Requirement |
|---------|-------------|
| Door width | 32" clear minimum |
| Threshold | 1/2" max |
| Door hardware | Lever or push/pull |
| Vestibule depth | 48" + door swing |
| Floor mats | Cannot impede |

### Interior Requirements
| Feature | Requirement |
|---------|-------------|
| Aisle width | 36" minimum |
| Turning radius | 60" at ends |
| Counter height | 36" max (portion) |
| Reach | 15"-48" AFF |
| Fitting room | 1 accessible per 6 |

### Checkout
| Feature | Requirement |
|---------|-------------|
| Counter section | 36" max height |
| Counter length | 36" minimum |
| Knee space | 27" high, 30" wide |
| Writing surface | Optional |

|--------|-------|-----|

## Code Considerations

### Occupancy Classification
| Use | Classification |
|-----|----------------|
| Retail sales | M (Mercantile) |
| Restaurant | A-2 (Assembly) |
| Small restaurant (< 50) | B (Business) |

### Occupant Load Factors
| Use | SF/Person |
|-----|-----------|
| Retail (standing) | 30 SF |
| Retail (merchandise) | 60 SF |
| Restaurant (dining) | 15 SF |
| Restaurant (standing) | 5 SF |

### Egress Requirements
| Size | Exits Required |
|------|----------------|
| < 50 occupants | 1 exit |
| 50-500 occupants | 2 exits |
| 501-1,000 occupants | 3 exits |
| > 1,000 occupants | 4 exits |
