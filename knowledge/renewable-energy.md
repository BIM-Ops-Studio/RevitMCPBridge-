# Renewable Energy Systems

## Solar Photovoltaic (PV)

### System Types
| Type | Description | Application |
|------|-------------|-------------|
| Roof-mounted | On existing roof | Most common |
| Ground-mounted | On ground racking | Large systems, rural |
| Building-integrated (BIPV) | Replace building materials | Architectural |
| Carport/canopy | Over parking | Commercial, multifamily |
| Floating | On water bodies | Reservoirs, ponds |

### Module Technologies
| Type | Efficiency | Cost | Notes |
|------|------------|------|-------|
| Monocrystalline | 18-22% | $$$ | Most common, best efficiency |
| Polycrystalline | 15-17% | $$ | Lower cost, less efficient |
| Thin-film | 10-13% | $ | Flexible, low efficiency |
| Bifacial | 20-25% | $$$$ | Collects light both sides |
| PERC | 20-23% | $$$ | Enhanced rear surface |

### System Sizing
| Building Type | Watts/SF Roof | Notes |
|---------------|---------------|-------|
| Residential | 4-5 W/sf | Based on available roof |
| Commercial | 10-15 W/sf | Flat roof, optimal tilt |
| Industrial | 15-20 W/sf | Large flat roofs |

### Roof Requirements
| Factor | Requirement |
|--------|-------------|
| Structural capacity | 3-5 PSF additional |
| Age | < 10 years remaining (or reroof) |
| Orientation | South preferred (±45°) |
| Shading | < 10% annual shading |
| Access | 3' perimeter, pathways |
| Fire setbacks | Per local fire code |

### Electrical Components
| Component | Function |
|-----------|----------|
| Modules | Generate DC power |
| Inverter | DC to AC conversion |
| Racking | Mounting structure |
| Combiner box | Aggregate strings |
| Disconnect | Safety isolation |
| Meter | Production monitoring |
| Rapid shutdown | NEC 2017+ safety |

### Inverter Types
| Type | Application | Efficiency |
|------|-------------|------------|
| String inverter | Residential, small commercial | 97-99% |
| Microinverter | Shaded roofs, complex | 95-97% |
| Power optimizer | Partial shading | 98-99% |
| Central inverter | Large commercial | 97-99% |

## Net Zero Design

### Definition Levels
| Level | Description |
|-------|-------------|
| Net Zero Site Energy | On-site generation = consumption |
| Net Zero Source Energy | Accounts for grid losses |
| Net Zero Energy Cost | Energy bill = $0 |
| Net Zero Carbon | No carbon emissions |

### Design Strategies
| Priority | Strategy | Impact |
|----------|----------|--------|
| 1 | Reduce loads | 30-50% reduction |
| 2 | Passive design | 10-30% reduction |
| 3 | Efficient systems | 20-40% reduction |
| 4 | Renewable generation | Offset remaining |

### Load Reduction Targets
| Building Type | Target EUI (kBtu/sf) |
|---------------|----------------------|
| Office | 25-35 |
| School | 25-40 |
| Retail | 30-45 |
| Multifamily | 25-40 |
| Single-family | 15-25 |

### Passive Strategies
| Strategy | Benefit |
|----------|---------|
| Orientation | Optimize solar gain |
| Glazing ratio | Balance daylight/heat gain |
| Thermal mass | Moderate temperature swings |
| Natural ventilation | Reduce cooling loads |
| Daylighting | Reduce lighting loads |
| Shading | Control solar heat gain |

## Geothermal Heat Pumps

### System Types
| Type | Description | Application |
|------|-------------|-------------|
| Vertical closed loop | Bore holes | Limited land area |
| Horizontal closed loop | Trenches | Adequate land |
| Pond/lake | Submerged loops | Water body available |
| Open loop | Well water | Good water quality |
| Standing column | Single deep well | Specific geology |

### Sizing Guidelines
| Factor | Guideline |
|--------|-----------|
| Bore depth | 150-400 ft per ton |
| Bore spacing | 15-20 ft apart |
| Horizontal loop | 400-600 ft per ton |
| Pond loop | 300-500 ft per ton |

### Performance
| Metric | Value |
|--------|-------|
| Heating COP | 3.5-5.0 |
| Cooling EER | 14-25 |
| Life expectancy | 25+ years (loop: 50+) |
| Energy savings | 30-60% vs conventional |

### Costs
| Component | Cost per Ton |
|-----------|--------------|
| Vertical bore | $3,000-5,000 |
| Horizontal loop | $1,500-3,000 |
| Heat pump unit | $1,500-2,500 |
| Total installed | $15,000-30,000 |

## Wind Energy
